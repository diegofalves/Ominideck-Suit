# Domain Glossary (OTM Migration Context)

- `MIGRATION_PROJECT`: aggregate root for migration scope and lifecycle.
- `MIGRATION_GROUP`: logical execution grouping inside a migration project.
- `MIGRATION_ITEM`: migration unit linked to group, object type, and deploy type.
- `OTM_OBJECT`: record returned from extraction queries.
- `deployment_type`: deployment strategy bound to table eligibility catalog.
- `DONE`: valid only when all previous required phases are complete.

Use these terms in SQL-related explanations to keep language consistent with OmniDeck.
