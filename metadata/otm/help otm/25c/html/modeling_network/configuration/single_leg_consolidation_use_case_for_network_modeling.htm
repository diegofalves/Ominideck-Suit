Single Leg Consolidation Use Case for Network Modeling

Different options are available for setting up logistics network modeling based on the scenario that you want to model. This use case will help you analyze the cost impacts on various single leg consolidation scenarios using parameter overrides. We'll create 3 modeling scenarios to compare costs based on consolidation as follows:

  • No Consolidation: All orders go direct. Disable bundling, same source and destination pair consolidation, and multistop.
  • Allow Consolidation but No Multistop: Allow consolidation of orders that have same source and destination location and bundling, but do not allow multistop consolidation.
  • Allow Consolidation with Multistop: Allow bundling, same source/destination pair and multistop consolidation.
