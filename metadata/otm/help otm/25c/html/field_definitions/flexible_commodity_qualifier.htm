Flexible Commodity Qualifier

Flexible Commodity Qualifier and Commodity Code fields describe the content of your shipment in terms of a specific classification. For example, you can classify the content of the shipment by industry-standard classifications such as NMFC, STCC, HTS, and SITC. If you use one of these standard classifications, enter the appropriate classification value in the Commodity Code field. Oracle Transportation Management contains a library of values for each industry-specific classification in the Item Classification Power Data.

The options are:

  • Classification: to limit the search to rates compatible with a specific User Defined Classification List ID.
  • Commod: to limit the search to rates compatible with a specific commodity.
  • Export HTS: to limit the search to rates compatible with a specific HTS code
  • HTS: to limit the search to rates compatible with a specific HTS code.
  • IATA SCR Code: to limit the search to rates compatible with a specific IATA SCR Code.
  • Item: to limit the search to rates compatible with a specific Item ID.
  • NMFC CLASS: to limit the search to rates compatible with a specific National Motor Freight Classification (NMFC) code.
  • SITC: to limit the search to rates compatible with a specific SITC code.
  • STCC: to limit the search to rates compatible with a specific STCC code.
  • User Defined Commodity: to limit the search to rates compatible with a specific User Defined Commodity ID.
