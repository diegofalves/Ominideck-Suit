User Favorites

This page is accessed via Configuration and Administration > Preferences > User Favorites.

Favorites are the records that, for quick reference, you select to appear in a pop-up before you see the search page. You can define favorite records for many of the tables in the database including order, shipment, and payment search results.

For example, you can create a favorite for locations. In that favorite you specify the top 10 locations which you use on a regular basis. Then whenever you use the Search () button the system fist displays those 10 locations. If you want to find a different location you are given the option to search for more locations via Find More.

After selecting favorites, you can still add additional search criteria before clicking Search.

Note: Favorites preferences apply only to the currently logged in user.

The table name for which you are modifying favorite records appears at the top of the page. The current favorite IDs appear in the grid below the table name.

Adding a New Favorite

From the User Favorites Page

  1. Click New Favorite.
  2. Choose a Table Name from the list.
  3. Enter an ID and click Save. The type of ID available changes depending upon which table name you selected. Repeat this step for additional records that you want as favorites.
  4. Click Save.
  5. Click Save Favorites on the User Preferences/Favorites page.

Note: After saving your new favorites, you must click Save Favorites for them to take effect.

From Other Pages

Many pages have a Add to Favorites/Remove from Favorites star icon next to object IDs. These pages include managers, power data, finder results, business object confirmation pages, etc. Use these icons to add or remove the object as a user favorite.

Note: You can also save any business object as a Favorite by clicking on the Add to Favorites icon anywhere that you see one in the UI.

Deleting Favorites

From the User Favorites Page

Deleting Favorites for an Entire Table

  1. To delete a "table" from the list, click Delete next to the far right of the table in the grid.
  2. Click Delete.
  3. Click OK in the pop-up or Cancel to close without deleting.
  4. Click Save Favorites at the bottom of the page.

Delete a Single Favorite from within a Table

  1. To delete an "ID" from the list, click Edit next to the far right of the table in the grid.
  2. Click Delete.
  3. Click OK in the pop-up or Cancel to close without deleting.
  4. Click Save at the bottom of the page.
  5. Click Save Favorites at the bottom of the page.

Note: After saving your new favorites, you must click Save Favorites for them to take effect.

From Other Pages

Many pages have a Add to Favorites/Remove from Favorites star icon next to object IDs. These pages include managers, power data, finder results, business object confirmation pages, etc. Use these icons to add or remove the object as a user favorite.

Note: You can also save any business object as a Favorite by clicking on the Add to Favorites icon anywhere that you see one in the UI.

Favorites Stars

A yellow star () indicates the record is a favorite. A gray star indicates the record is not a favorite.

Adding a Favorite

  • Click the Add to Favorites icon () to add that record as a user favorite.

Deleting a Favorite

  • Click the Remove from Favorites icon () to remove that record from user favorites.

Viewing Favorites

You can view current favorites via:

  • Favorites pop-up available from the Unified Global Header
  • User Favorites page
  • Favorites column on finder results pages
