Configuration and Administration

Public Menus

Oracle Transportation Management provides the following public menus that you can use as a starting point for designing additional user menus. If you want to start with one of these menus, you can copy it and modify the structure.

Use the Menu Manager (Configuration and Administration > User Configuration > Menu Manager) to view the default structure of these menus.

Note: All PUBLIC menus have been migrated and should be copied and edited using the Menu Manager instead of the legacy Menu Editor.

DEFAULT Level Menus

The following menus are assigned by default to users assigned to the DEFAULT level. The menus appear in the order listed below:

  • DEFAULT_MENU: This is the Oracle Transportation Management menu.
  • GTM_MENU: This is the Global Trade Management menu.
  • ASK_ORACLE_MENU: Contains links to all Ask Oracle pages.
  • HELP_MENU: Contains help links.

ADMIN Level Menus

The following menus are assigned by default to users assigned to the ADMIN level. The menus appear in the order listed below:

  • ADMIN_MENU: This is the Oracle Transportation Management ADMIN menu.
  • GTM_ADMIN_MENU: This is the Global Trade Management ADMIN menu.
  • ASK_ORACLE_MENU
  • HELP_MENU

DBA.ADMIN User Menus

The following menus are assigned by default to DBA.ADMIN users. The menus appear in the order listed below:

  • DBA_ADMIN_MENU: This is the Oracle Transportation Management DBA.ADMIN menu.
  • GTM_DBA_ADMIN_MENU: This is the Global Trade Management DBA.ADMIN menu.
  • ASK_ORACLE_MENU
  • HELP_MENU

SERVPROV.ADMIN User Menus

The following menus are assigned by default to the SERVPROV.ADMIN user. The menus appear in the order listed below:

  • SERVPROV_ADMIN_MENU is the default menu for the service provider administrator user.
  • HELP_MENU

SERVPROV User Menus

The following menus are assigned by default to any service provider that you add to the SERVPROV domain. The menus appear in the order listed below:

  • SERVPROV_MENU is the default menu that is used for any service provider that you add to the SERVPROV domain.
  • HELP_MENU
