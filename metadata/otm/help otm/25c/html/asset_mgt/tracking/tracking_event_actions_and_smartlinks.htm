Tracking Event Actions and SmartLinks

The Tracking Event manager is accessed via

  • Fleet Management > Visibility > Tracking Event
  • Shipment Management > Visibility > Tracking Event

Click Actions on the Tracking Event page to launch the following actions.

Action

Description

Add Related Event

Adds a related tracking event using the Tracking Event Manager.

Attach Documents

This action allows you to assign one or more unowned documents to the business object you are working with.

Generate Document

Use this action to Validate and generate a report-based document for the selected business object.

Limited Documents

This action searches for documents within a business object by using a finder filter.

Upload Document

This action allows you to browse your local area network and upload a document to an Oracle Transportation Management object or a Global Trade Management object.

Copy Tracking Event

Creates a new tracking event by copying an existing event and then letting you edit it.

Mark as Error

Changes the processing status to MARKED AS ERROR and removes all links to the equipment, driver, power unit, shipment, and tracking event.

Mark for Purge

Marks selected tracking event(s) for data purge. Â 

Reprocess

Removes all the links to equipment, drivers, power units, shipments, and the related tracking events. The processing status is set to REDO.

Send Interface Transmission

Set Indicator

SmartLinks

Right-click on any ID to launch SmartLinks.

SmartLink

Displays

Documents

 This opens the Documents results page showing the Document IDs that are linked to this tracking event.

View Related Events

Displays an event related to the current one. An optional event date range is defined in the glog.trackingevent.viewEvents.rangeInDays property.
