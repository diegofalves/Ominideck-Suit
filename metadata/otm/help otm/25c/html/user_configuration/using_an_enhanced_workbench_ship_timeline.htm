Using an Enhanced Workbench Shipment Timeline

This page is accessed via Configuration and Administration > User Configuration > Enhanced Workbench. Select an existing layout. The Workbench Layout list will show both enhanced and legacy layout IDs. The legacy layout IDs are marked with an asterisk. While these layouts can be viewed in the Enhanced Workbench, they cannot be edited. Any legacy layouts that you view in the Enhanced Workbench uses all of the enhanced layout icons. You can convert legacy layouts to enhanced layouts using the Workbench Manager actions.

After you design layouts for a workbench, you can use the workbench by selecting the layout from the drop-down list.

When paired with a buy shipment table, the shipment timeline shows any tracking events entered for a buy shipment.

Icons and Buttons

  • (Lock View): By default, the lock icon is shown as open (). When you click the lock icon to lock a pane, the icon is shown as closed (). Click the Lock View icon to lock that pane and keep the current shipment timeline view; then, if you click another shipment the displayed shipment timeline does not change. This is useful when you have two shipment timeline regions, because you can select a shipment to view and then lock that shipment timeline pane. With one pane locked, you can select a second shipment to view in the second pane.
