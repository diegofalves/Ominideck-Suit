Purchase Orders Tab

The Purchase Orders tab provides a view of various data including purchase orders by:

  • Month of Import,
  • Importing Country,
  • Transport Mode,
  • on hold for
    • Production Classification
    • Control Screening
    • Sanction Screening
    • Restricted Party List Screening
  • by control information such as
    • Authorization
    • Control Code
    • License Code
    • Exception Code

It also includes Extended Amount by

  • Month of Import,
  • Importing Country, and
  • Transport Mode
