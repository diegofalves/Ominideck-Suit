Declarations - Imports Tab

The Declarations - Imports tab provides a view of various data including:

  • Import declarations by
    • month of import
    • declaration type
  • Import declaration lines by
    • month of import
    • declaration type
  • Customs value base by
    • month of import
    • declaration type
  • Cost Breakdown by
    • month of import
    • declaration type
