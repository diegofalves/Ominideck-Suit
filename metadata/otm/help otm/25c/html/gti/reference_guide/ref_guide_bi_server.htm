Global Trade Management Subject Area

The Global Trade Intelligence subject area is structured into multiple analysis folders one for each Global Trade Management business object (trade transaction, trade transaction line, etc.). Each analysis folder then includes one facts folder and multiple dimensions folders. It is important to note that only compatible dimensions (and columns underneath) are available in each analysis folder. All columns under the respective facts and dimensions folders are also functionally grouped under headers (identifiers, cost, distance, flex fields, etc.) for ease of identification and usage.

Note: While performing ad-hoc queries or building customer-defined reports end users are warned to not use dimensions or columns across analysis folders as these may lead to query errors due to incompatibility.

GTI Dashboard Reports

This is the default Global Trade Intelligence (GTI) dashboard (accessible to typical Oracle Transportation Management and Global Trade Management users) and contains reports covering the key business areas under respective tabs. This dashboard is available as a preferred dashboard in the OTM/GTM user preferences.

This main GTI Dashboard report is located in the catalog in /shared/GTI Reports/_portal/GTI Dashboard

The GTI dashboard has the following tabs:

  • Purchase Orders
  • Sales Orders
  • Shipments - Export
  • Shipments - Imports
  • Declarations - Exports
  • Declarations - Imports

Detailed Reports

The following section explains some of the details found across tabs in the GTI dashboard.

Export_GTI Transaction Line Count by License Code Report

This report shows the distribution of the transaction line count by the assigned license codes. This report is located in the catalog in /Shared Folders/GTI Reports/GTI Dashboard Reports/Export_GTI Transaction count by License Code

Import_GTI Declared Value by Transport Mode Report

This report shows the distribution of the declared value of transactions by transport mode. This report is located in the catalog in /Shared Folders/GTI Reports/GTI Dashboard Reports/Import_GTI Declared Value by Transport Mode.
