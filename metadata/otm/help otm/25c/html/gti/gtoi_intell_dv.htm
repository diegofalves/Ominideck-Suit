Restricted Party

This project is located in theÂ catalog in Shared Folders/GTOI Data Visualization/Restricted Party.

The DV project contains the following graphs:

Restricted Party Screening

This visualization shows the restricted party screening results based on the selected country, city, and year.

Screening Status by Date

This graph shows the number of parties screened and their statuses (like approved, escalated, rejected, requires review) based on the date criteria you select (like a specific date, a date range, or relative time).

The Screening Status by Date table displays detailed information about the screened parties including their first and last name, company name, alternate name, matched factor, etc.Â 

Using as the Home Landing Page

You can create an enhanced workbench using this DV project and set it to show instead of the springboard. This section contains detailed steps to turn your home page into a spotlight of your business.

Setting up the DV Project

  1. Login with any user which has a BIADMINISTRATOR or a BIAUTHOR Business Intelligence role set in OTM.
  2. Go to Global Trade Intelligence > Catalog.
  3. In the Folders pane, go to Shared Folders > GTOI Data Visualization.
  4. Click on the "Restricted Party" DV Project and in the Tasks pane, select Copy.
  5. In the Folders pane, go to Custom and in theÂ Tasks pane, click Paste. The DV project is copied to the Custom folder where you can edit it and make changes. Also in this location, the project is not overwritten during an upgrade.

Note:
1. By default, DBA.ADMIN has access to the Global Trade Operational Intelligence subject area.
2.Â A user can access the Global Trade Operational Intelligence subject area to create a new workbook when GTOI is added as a business intelligence application. You can add this application via the User Manager page.
2.Â After the changes, customer can take a back-up of the updated project by using the Archive utility.

Creating the Workbench

Next, you create an enhanced workbench layoutÂ that contains your Restricted Party DV project.

  1. Go to Configuration and Administration > User Configuration > Enhanced Workbench.
  2. Click Create Layout.
  3. Enter a Layout ID.
  4. Select the WORKBENCH DEFAULT logic configuration.
  5. Select the Auto Refresh checkbox and set the Auto Refresh Interval to 5 minutes.
  6. Click OK to create the layout.
  7. To define a new pane, hover over the Layout Configuration icon to see the content editor menu and click Add Content.
  8. Select a Component Type of Data Visualization.
  9. Enter a Tab Name and Project PathÂ (for example, /@Catalog/shared/Custom/Restricted_Party). And click OK.
  10. Â Click INSERT ICON (Done Editing).

Setting as the Home Landing Page

Finally, you set your user preference to use the new workbench layout as your home page.

  1. Go to Preferences > User Preferences.
  2. Edit the user preference that is used by your domain.
  3. Select a Name of Home Workbench Layout ID.
  4. Select a Value of the workbench layout ID you created above.
  5. Click Save and Finished.
  6. Sign out and then back into OTM to apply the user preference. When you log in, you see that the Restricted Party workbench is now your home page.
