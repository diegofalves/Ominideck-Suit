Mass Update

Note: This popup only displays if your screen set is configured to allow edits. Then, you can click the Mass Update icon to edit fields on a Finder results page or Workbench table.

This page allows you to make multiple edits at one time for one or more columns, in one or more rows directly from a Finder results page or Workbench table. Select one or more rows to edit and click the Mass Update icon. The Mass Update page opens.

Note: The information displayed on this page varies depending on your configuration and the rows selected.

In the IDs section, you see the ID for the selected rows.

In the Fields section, you see all the editable fields. You can enter one value per field, or clear the checkbox so no value appears. Change as many fields as necessary. When finished, click Save to save your changes and return to the Finder results page or Workbench table or click Cancel to disregard your changes and return to the Finder results page or Workbench table.

Note: The Save dialog box is not shown when using mass update in a Workbench.

A Save dialog box is displayed with the list of rows to be saved. Each row is saved individually. As each row is saved, the object is locked. Once saved, the object is unlocked. The rows in this table can look as follows:

  • Bold Text: Record is being saved.
  • Green Text and icon:Â Record saved successfully.
  • Red text with and icon: There is an error. Click the Expand icon for details on the error.
  • Text crossed out and icon: The record was skipped because the Cancel button was selected.

The Cancel button stops processing and any rows after the current row will not be processed. Records that were not processed are indicated by the text being crossed out and the Cancelled icon. The Cancel button is disabled once clicked or when all rows have been processed.

The Close button is enabled once all rows have been processed.
