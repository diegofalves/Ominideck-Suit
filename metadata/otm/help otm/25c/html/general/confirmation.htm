Business Object Confirmation

When you click Finished after creating or editing any business object, Oracle Transportation Management saves the object and displays this confirmation that the object was successfully created or modified.

Note: If for some reason, Oracle Transportation Management could not add the new record, the Success page displays a message on the Error tab. You can view the Advanced to try and troubleshoot the error or contact your Oracle Transportation Management System Administrator.

For each object, you can view or edit the details. For some objects, you can also perform actions that are specific to that business object. These actions are the same as those that would appear on the Results page for the business object.

Favorites Stars

A yellow star () indicates the record is a favorite. A gray star indicates the record is not a favorite.

Adding a Favorite

  • Click the Add to Favorites icon () to add that record as a user favorite.

Deleting a Favorite

  • Click the Remove from Favorites icon () to remove that record from user favorites.

View Details

Click View to display summary data for the object. You can also perform actions, edit or create a new object.

Edit

Click Edit to display the first data-entry page for the object ready for editing.

View Log

Click View Log to display the System Log Query page in the Process Manager.

View Explanation

Click View Explanation to display the Explanation Viewer that can be used to diagnose algorithm behavior, report errors, trace agent activity or report on the success and failure of a background process.

Note: This button appears only if you have turned on Explanations.

Create Another

Click Create Another to create a new object of the same type.

Modify Another

Click Modify Another to display the Search page and find additional objects for editing or review.

Perform Actions

To perform an action, mark the checkbox next to the object that you want to perform and select the action from the list.
