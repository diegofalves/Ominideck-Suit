Inline Edit

For keyboard only users

Use the Tab and arrow keys to navigate to an editable field. Once focus is on an editable field a icon (Edit) appears to the right of the field. Press Enter to open the Inline Edit pop-up and make your changes as discussed in the Inline Editing section of the Search Results help topic.

Skip Repeated Navigation

On all pages with the Unified Global Header, the first tab stop on the OTM/GTM page opens a skip navigation list that allows you to skip the global header and jump straight to the main page. Also, press ALT + 1 to access skip navigation at any time.

Note: To use skip navigation on most pages, use the display option of Replace Current Window when creating, viewing, or editing an object via the finder results.

The skip navigation list contains the following links:

  • Home: Skip to the OTM springboard.
  • Skip to Content: If there is no focusable item before the current active tab stop, skip to the current active tab stop. Or, skip to the default focusable item on the page or to the first item in the springboard.
  • Skip to Search: Skip to the Global Search field in the global header.
  • Skip to Tab: You can either select the Skip to Tab link or select a tab from the list of available tabs. When you select Skip to Tab, skip to the first focusable item inside the currently active tab. If you select a tab from the list, skip to that tab and press Enter to move focus into that tab.
  • Skip to Footer: Skip to the first button in the footer or the last clickable link on the page.
