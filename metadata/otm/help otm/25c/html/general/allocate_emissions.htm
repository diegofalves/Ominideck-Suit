Allocate Emissions

This page is accessed viaÂ Shipment Management > Shipment Management > Buy Shipments or Sell Shipments > Actions > Shipment Management > Emissions > Allocate Emissions.

Allocate Emissions is used to divide the emissions of a shipment among its order releases or order release lines, by distributing the emissions according to a specified allocation metric.

You need to define the parameters for allocating emissions using the Emission Allocation Rule Manager. For example, emissions can be allocated based on metrics such as weight carried distance. The application will select emission allocation rules from the emission activity rule, which is used to generate the corresponding emission activity. When the emission activity does not have any emission activity rule, then the application will use the default emission allocation rule.
