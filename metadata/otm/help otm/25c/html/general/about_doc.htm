Online Help

The online help is the Oracle Transportation Management user/reference manual that consists of books containing HTML help topics describing the functionality of the product.

Note: Help is documented based on the standard product. You may have a configured version so fields may be hidden, tabs may be hidden, etc. Help will not reflect those changes.

Online Help Conventions

Online help can be accessed from the Navigation menu or in context with a specific page in the software using the Help icon at the top section of any page. The Help navigation menu contains the following components:

  • Table of Contents structured similarly to the standard menu in Oracle Transportation Management.
  • Index provides a search engine so you can locate topics based on keyword searches.
  • Search does a full text search of the word or words entered. If searching for multiple words together, you can put quotation marks around the words to limit the number of results returned.
  • Glossary provides definitions of common logistics terms. Index and Search do not find Glossary entries. Additionally, glossary terms cannot link to other topics. Index and Search should be used before Glossary.
  • Favorites allows you to set topics you are viewing as a favorite, so you can easily find it again. This is limited by machine and browser.

Help pages usually contain one or more of the following navigation aides.

  • Navigation links at the top of the page (in the blue bar) that jump to specific sections on the page.
  • Hyperlinks navigate to other topics that provide additional detail.
  • Steps provide straightforward, numbered procedures.
  • Related Topics provides links to topics that are related to the topic you are reading, in addition to links within the topic.
  • icon pages (backwards or forwards depending on the direction of the arrow) through related help topics that can be read in a logical order. These are available for a couple series of related topics that are considered books in the Table of Contents under How To/Configuration topics: 'Restricted Party Screening' and 'Bulk Plan Performance and Tuning'. To use these, the left pane of Help, must be visible. If it does not display, click Show Contents.

  • Note: Notes indicate a special point of interest, tip, or warning.

PDFs

There are some PDFs that are available as part of help. These are not true documents that are released as PDFs but rather a series of help topics that can be read in order. With normal help topics, if you want to print them, you have to print each topic individually. With these topics, there are compiled into a PDF and you can print it. Since these are a series of help topics, when you print it, you get all the footers. If you print it, the links to other help topics will not work. There are PDFs for Bulk Plan Tuning and Performance, Restricted Party Screening,Â Order Management, and Landed Cost Simulator. These are found in the How To/Configuration Topics in the help Table of Contents and can be accessed from the Related Topics section of relevant topics.
