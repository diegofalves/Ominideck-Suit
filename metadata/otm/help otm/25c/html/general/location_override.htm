Brokerage and Forwarding

Location Content Override

You can override specific content for any location on a shipment, freight forwarding order release, or for involved parties on a job to ensure that location information is kept consistent across multiple systems to create accurate documentation for international shipments. A content override does not change the location assignment on the business object; only specific location data such as address information, and location name, for example.

The override information is transactional in that it only applies to the business object on which you record the override. Override data does not impact master data stored for each location in the Location Manager. You can only override location content when a location field is populated.

  1. When you click Set Location Overrides, a secondary page may appear (depending on the business object on which you are working) listing location IDs for which you can provide override information.
  2. Click Create Override next to a location to display the Location Content Override page that displays the fields that you can override. Each field is automatically pre-populated with the master data recorded for the location.
  3. Enter the override data and click Finished. An Edit Override button appears next to any location that has existing override data.

Involved Parties

Involved Party grids for an order release, shipment and job also display an override button that allows you to modify specific location data for the location that is assigned to the contact identified as the involved party. Contacts are assigned to a location in the Contact Manager or in the Location Manager.

Note: You can only override location content if the involved party contact is assigned to an existing location.
