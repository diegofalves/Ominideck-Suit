Upload Mail and User-Defined Images

This page is accessed via Configuration and Administration > Branding > Upload Mail and User Defined Images. It can only be accessed by someone with DBA access.

Use this page to upload mail and user-defined images. You can upload no more than 20 files at a time.

Mail Images

Mail images are used by notification stylesheets. These files will be uploaded to the /images/<glog.customPath>/mail directory.

The logo is manipulated by the glog.mail.logo.* properties. The relevant properties are:

  • glog.mail.logo.filename: Filename of the mail logo image
  • glog.mail.logo.height: Height of the mail logo image
  • glog.mail.logo.width: Width of the mail logo image

Other usages of mail images include user-defined notifications that include new images. These images should be uploaded here.

User-Defined Images

The user defined images can be accessed by creating images using the Images power data page.

These images can then be used by the Set Images action and viewed when adding the User Defined Image as a column on the result page of a user-defined screen set. These files will be uploaded to the /images/<glog.customPath>/icons directory, where the values inside of <> are properties that set locations on a server.

Note: If you try to upload an image with the word "logout" in the file name, you will see an error message. Remove the word "logout" from the file name and upload the file again.

Uploading Images

  1. To upload an image, click the Browse button. Select the image file. You can select images in GIF, PNG, JPEG, TIFF, or mobile device WBMP formats. For example, .gif, .png, .jpg, .wbmp, and .tif.
  2. If you want to add multiple files, click the Upload Another File link after you specify the first image. There is a maximum of 20 files that you can upload at one time.
  3. Click OK.
