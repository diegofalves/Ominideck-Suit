Charter Voyage Stowage Mode Summary

Stowage Mode can be viewed as a "compartment" within the vessel on a voyage that describes how order content is stored. For example, you could create a stowage mode that identifies order content in containers, roll on/roll off, etc. For each Stowage Mode within a single charter voyage, there is a capacity associated with it. Capacity allocations are usually based on contractual agreements between a company and the sea carrier. The capacities on a single charter voyage (by stowage mode) are worked out by the carrier and defined on voyage here.

This page displays all of the stowage modes that have been created for this charter voyage. A Consol ID appears only after you create a consol for a stowage mode using the Create Consol action.

  1. The Source/Destination Terminal Location IDs appear if you have assigned these locations to a stowage mode on the Stowage Mode Details page.
  2. Click new charter voyage stowage to create a new stowage mode.
  3. You can also edit and delete records from this page.
