ORDER_BASE_LINE_VARIABLE_TRANSPORT_HANDLING_UNIT

Transport Handling Unit

Order Base Line

This Auto Assign Type allows you to assign an order base (OB) ship unit or line item Transport Handling Unit (THU) based on the THU associated with either the source location or destination location on the order.

A typical use case would be to assign an order base line or ship unit THU based on the THU associated with either the source or destination location on the order. On the Definition tab you would configure an OB Line Source/Dest THU when the OB Line THU is null.

SHIPMENT_REQUIRED_DOCUMENT

Standard Document

Shipment

Populates SHIPMENT_REQDOC, the standard document placeholder for shipments.

SHIPMENT_REQUIRED_DOCUMENT_REVIEWER

Reviewer

Standard Document on a Shipment

Populates SHIPMENT_REQDOC_REVIEWER, the list of reviewers associated with each standard document. Assign child records for a given shipment and document type. Rather than create an entire new business object for a shipment document, these types are based on the shipment but support shipment-level criteria and document type criteria. When the rule is applied, the shipment is searched for a matching document typeDocu. If no such standard document exists, the rule is skipped. Otherwise, specified child records are added to the shipment standard document.
