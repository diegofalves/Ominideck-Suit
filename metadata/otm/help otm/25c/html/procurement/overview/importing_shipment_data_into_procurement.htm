Importing Shipment Data into Sourcing

Before you can generate bid information, you must import shipping data into Oracle Transportation Sourcing and then build a shipment set. Since Oracle Transportation Sourcing will use this data to build shipping lanes, the import process should be run frequently to populate Oracle Transportation Sourcing with up-to-date information.

Shipping data can come from:

  • A local instance of Oracle Transportation Management
  • Other transportation management software (via a CSV file)

Importing CSV Data

You can import shipment data from non-Oracle Transportation Management software by preparing a data file in CSV format and using the CSV utility to bring the data into Oracle Transportation Sourcing. For more information, consult the Data Management Guide located on the Oracle Help Center.

You can import:

  • OTM shipments and use the Build Shipment Set action, or
  • Sourcing shipments and use the Assign Shipment Set action.

Importing Oracle Transportation Management Data

You can define the data to be pulled from Oracle Transportation Management using the Shipment Set Criteria page. After you have specified the selection criteria in your shipment set, run the Build Shipment Set action to pull shipments that meet that criteria.
