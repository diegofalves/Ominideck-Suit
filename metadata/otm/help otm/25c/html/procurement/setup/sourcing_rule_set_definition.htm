Sourcing Rule Set Definition

This page is accessed via Sourcing > Setup > Rule Set. Click the Sourcing Rule Set Definition tab.

The Sourcing Rule Set Definition tab allows you to fine tune rules that govern how the solver builds solutions.

Limit Awards by Number of Service Providers Rule

To activate this rule, enter values in the Minimum/Maximum Service Providers for Total Lanes and/or Minimum/Maximum Service Providers for Each Lane fields.

Limit Awards by Service Providers Rule

You can use this rule to guarantee the minimum or maximum amount of freight awarded to a specific carrier or group of carriers. For example, you can use this rule to model the following scenarios:

  • If you do not want to rely on a single carrier, you can use this rule to ensure that bids are awarded to more than one carrier.
  • If you already have minority carrier/contract commitments in place, you can use this rule to ensure that bids are awarded appropriately.

Note: You can enter multiple rules in this category.

Limiting Awards by Service Provider:

  1. Select one of the following from the Criteria drop-down list:
    • At Least
    • At Most
  2. Select one of the following from the Value Type drop-down list:
    • Percentage of Shipments
    • Count of Shipments
    • Monetary Spend
    • Percentage of Monetary Spend
  3. If you selected Percentage of Shipments, Count of Shipments, or Percentage of Monetary Spend, enter a corresponding value in the Percentage/Count field.
  4. If you selected Monetary Spend, type a corresponding value in the Monetary Spend field.
  5. Select a profile in the Service Provider Profile ID field.
  6. Select one of the following from the Scope drop-down list:
    • All Lanes
    • Each Lane
  7. Click Save.

Winning Service Provider Limit Rule

You can use this rule to require carriers to win a certain amount of business (the limit) in order to be awarded any business at all. You can use this rule:

  • To eliminate a carrier that does not offer competitive rates.
  • To reduce the potential overhead costs involved with doing business with a carrier, if that carrier handles only a small percentage of your business.

Note: You can enter multiple rules in this category.

Specifying the Limit:

  1. Select one of the following from the Value Type drop-down list:
    • Percentage of Shipments
    • Count of Shipments
    • Monetary Spend
    • Percentage of Monetary Spend
  2. Enter a corresponding value in the Percentage/Count or Monetary Spend field.
  3. Select one of the following from the Scope drop-down list:
    • All Lanes
    • Each Lane
  4. Click Save.

Exclude Packages Rule

Note: You can enter multiple rules in this category.

  1. Select a service provider in the Service Provider Profile ID field.
  2. Click Save.

Favor Service Provider Rule

Note: You can enter multiple rules in this category.

  1. Select Value Type of one of the following.
    • Percentage of Shipments
    • Count of Shipments
    • Monetary Spend
    • Percentage of Monetary Spend
  2. Enter a corresponding value in either the Percentage or Amount field.
    • For percentage, enter a positive value (e.g. 10) to lower the computed cost of the bid by 10% or a negative value (e.g. -10) to raise the computed cost of the bid by 10%.
    • For amount, enter a positive number (e.g. 20) to lower the computed cost of the bid by a flat amount of 20 or a negative amount (e.g. -20) to raise the computed cost of the bid by a flat amount of 20.

      Note: The amount is not a variable per mile.

  3. Select a profile in the Service Provider Profile ID field. The corresponding value type and percentage/amount will be applied for all service providers in the profile.
  4. Click Save.

Exclude Bids by Attributes Rule

Note: You can enter multiple rules in this category.

  1. Select one of the following from the Attribute Type drop-down list:
    • SERVPROV
    • BID
  2. Select a name from the Attribute Name drop-down list.
  3. Select a criteria from the Criteria drop-down list.
  4. Enter the attribute's value in the Value field.
  5. If applicable to your rule, define the attribute's duration in the Duration fields.
  6. Click Save.

Favor Bids by Attribute Rule

Note: You can enter multiple rules in this category.

  1. Select one of the following from the Attribute Type drop-down list:
    • SERVPROV
    • BID
  2. Select a name from the Attribute Name drop-down list.
  3. Select a criteria from the Criteria drop-down list.
  4. Enter the attribute's value in the Value field.
  5. If applicable to your rule, define the attribute's duration in the Duration fields.
  6. Select one of the following from the Favor Value Type drop-down list:
    • Amount
    • Percentage
  7. Enter the favored value in either the Favor Percentage or the Favor Cost field.
    • For favor percentage, enter a positive value (e.g. 10) to lower the computed cost of the bid by 10% or a negative value (e.g. -10) to raise the computed cost of the bid by 10%.
    • For favor cost, enter a positive number (e.g. 20) to lower the computed cost of the bid by a flat amount of 20 or a negative amount (e.g. -20) to raise the computed cost of the bid by a flat amount of 20.
  8. Click Save.
  9. Click Finished when you are done building rules.
