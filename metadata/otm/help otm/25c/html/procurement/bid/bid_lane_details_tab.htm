Bid Manager: Lane Details

This page is accessed via Sourcing > Bid Creation > Bid. Then click Lane Details.

On this tab, you enter lane and service provider information for the bid. The lane IDs and service provider IDs available for selection depend on the specified bid round ID and its associated objects such as project, lane summary, and constraint set.

To enter lane details:

  1. Enter a Lane ID. You can only select a lane ID that is included in the lane summary associated to the bid round ID specified on the Bid Header tab.

    Note: If yes is selected for the Restrict to Prior Bid Round Service Providers option and if the selected service provider only bid on some lanes in the prior bid round, then you will only see those lanes if you click f or l.

  2. Enter an Equipment Reference Unit ID. This field displays if the Bid By Equipment Reference Unit is selected on the lane summary.
  3. Enter an Equipment Group ID. This field displays if theÂ the Bid By Equipment GroupÂ is selected on the lane summary.
  4. Enter a Service Provider ID. You can only select a service provider ID that is included on the constraint set associated to the bid round ID specified on the Bid Header tab.

    Note: Once you save a bid, you cannot change the service provider by editing the bid. The Service Provider ID field will be read-only. This is to prevent rate offerings/rate records associated with the bid from breaking.

    Note: If yes is selected for the Restrict to Prior Bid Round Service Providers option on the project and if the selected service provider only bid on some lanes in the prior bid round, then you will only see those lanes if you click f or l.

  5. Click Bid Detail to view the bid details.
