Sourcing

View Bid Report

Note: This action is only available when logged into Oracle Transportation Sourcing as a service provider.

This page is accessed via Sourcing > Bid Creation > Bid > Actions > View Bid Report.

Since as service provider logging into Oracle Transportation Sourcing you do not have access to the bid round manager, the View Bid Report action is only available from the bid manager.

The bid report allows you, a service provider, to double-check if the bids uploaded via the carrier response template were properly saved. The bid report is an MS Excel spreadsheet and includes the same columns as the carrier response template which are populated with the data uploaded via the Upload Bid Response action. In addition, the bid report contains the following fields:

  • Computed Cost
  • Bid ID
  • Status: This is the rate loading status and has the possible values of RATING_NOT_STARTED, RATING_FAILED, and RATING_COMPLETED.
  • Message

Viewing a Bid Report

  1. Select a bid and click Actions > View Bid Report.
  2. By default, the Bid Round ID field is auto-populated with the bid round ID associated with the selected bid.

    Note: You should not change the bid round ID. Changing the bid round ID can cause inconsistencies in the bid report.

  3. Select the Display All Lanes checkbox to display all lanes. If selected, lanes with bids are shown first then lanes without bids are shown. If not selected, only lanes with bids are shown. By default, the Display All Lanes checkbox is selected.
  4. If you want to run the action in the background, accept the default of Yes for the Run Job in Background field. The server can time out while waiting for a result if the action is run in the foreground for a large data set. Hence, for large data sets, you need to run the action in the background.
    If you selected Yes for Run Job in Background, an email with the bid report attached is sent to the sourcing contact listed on the service provider.
    If you selected No for Run Job in Background, you will be asked whether you wish to save the bid report to your hard drive or open it in Excel.
  5. Click View Bid Report.
