Bid Manager: Bid Header

This page is accessed via Sourcing > Bid Creation > Bid.

The Bid manager allows you to enter bids or modify details on existing bids. Each bid must be assigned to a bid round.

In most cases, the majority of bids will be uploaded to Oracle Transportation Sourcing via the carrier response template. A service provider/carrier could then log in using their SERVPROV user name and password to view and edit their uploaded bids.

Entering Bid Header Information

  1. Enter a Bid ID.
  2. Enter a Bid Round ID. The bid round ID specified here will determine what information is available for selection or automatically populated on the Lane Details and Bid Detail tabs.

    Note: If you click search, you only see bids rounds that are open and have a CRT Configuration ID specified on them.

  3. Select the domain for this bid from the Domain Name drop-down list.
  4. Click Lane Detail to go to the Lane Details tab.
