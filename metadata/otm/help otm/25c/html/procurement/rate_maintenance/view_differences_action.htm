View Differences Action

This Approval Summaries Detail workbench layout is accessed as follows:

  1. Navigate toÂ Sourcing > Rate Maintenance > Approval Summary. Enter search criteria and click Search.
  2. Select Always Open in New Window so that the approval summary page opens in a pop-up window.
  3. Right-click on a row in the search results and select View Approval Details. The APPROVAL_SUMMARY_DETAIL workbench layout open in a pop-up window.
  4. On the Approval Details page, select either a rate offering or rate record row.
  5. Click Actions > View Version History.
  6. Select a rate offering or rate records and click theÂ View Differences icon.

Note: You can only view differences if the selected rate offering/rate record has a parent ID.

Use this action to view the differences between a rate offering or rate record and the previous version (parent). You can view the details of previous versions and compare them to the selected rate offering/rate record.

Data is only shown on this page if the values for a particular field, element, or object are not identical between the select rate and the parent rate. The View Differences page shows the values for both the selected rate and the parent rate which allows you to compare the two sets of values.

This page contains 3 columns:

  • Rate Offering/Rate Record Field: This column contains the field, element, or object name.
  • Rate Offering/Rate Record ID <selected rate offering/rate record ID>: This column contains the value from selected rate offering/rate record.
  • Parent Rate Offering/Rate Record ID <ID of the parent rate offering/rate record for the selected rate offering/rate record>: This column contains the value from the parent rate offering/rate record.

For Rate Offering

The View Differences page for a rate offering may contain the following sections depending on what differences there are between the rate offering and its parent. The following possible sections are only shown if the data on the select rate offering differs from that on the parent rate offering and include:

  • Rate Header: Shows the non-identical rate offering and rate version data.
  • Stop-off Charge: Shows the non-identical stop off charges. The data in the Rate Offering Field displays "Low Stop # High Stop #" (for example, Low Stop 8 High Stop 10). The other columns contain the values.
  • NMFC: Shows the non-identical NMFCs. The Rate Offering Field column displays "NMFC Low # NMFC High #" (for example, NMFC Low 70.0 NMFC High 100.0). The other columns contain the values.
  • SMC: Shows the non-identical SMC discounts. If SMC discount is not included in either the selected rate offering or the parent rate offering, this section is not displayed. The Rate Offering Field column displays Min Discount, L5C Discount, etc.. The other columns display the values.
  • Accessorials: Shows non-identical accessorial charges. All columns show the Accessorial Code ID.
  • Special Services: Shows the non-identical special service accessorial charges. All columns show the Special Service Code ID/Accessorial Code ID combination.
  • Involved Parties: Shows the non-identical involved parties. The Rate Offering Field column displays Involved Party Contact ID. For the selected rate offering and the parent rate offering, the involved party information displays "Communication Method <METHOD> Involved Party Qualifier <QUALIFIER>.
  • Remarks: Shows the non-identical remarks. The Rate Offering Field column displays the Remark ID. For the selected rate offering and the parent rate offering, Remark text is shown.

For Rate Record

The View Differences page for a rate record may contain the following sections depending on what differences there are between the rate record and its parent. The following possible sections are only shown if the data on the select rate record differs from that on the parent rate record and include:

  • Attributes: Shows the non-identical the rate record attributes and lane data.
  • Reference Numbers: Shows the non-identical reference numbers based on reference number qualifier ID.
  • Stop-off Charge: Same as rate offering stop-off charge comparison above.
  • NMFC: Same as rate offering NMFC comparison above.
  • SMC: Same as rate offering SMC Discount comparison above.
  • Base Rate Cost: Lists the non-identical base rate costs in sequence. The first column is alwaysÂ blank. The second and third columns list the conditions and costs.
  • Accessorials: Same as rate offering accessorial comparison above.
  • Special Services: Same as rate offering special services comparison above.
  • Remarks: Same as rate offering remarks comparison above.
