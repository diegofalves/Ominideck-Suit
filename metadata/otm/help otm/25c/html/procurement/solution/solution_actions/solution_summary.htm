Solution Summary

This page is accessed via Sourcing > Bid Analysis > Scenario > Actions > Solution Summary.

This action gives you a formatted summary of the selected solution. Run this action after either solving the scenario via the Sourcing solver or manually creating a scenario/solution.

Note: When adding bids or solution detail records manually, the following information: %Lanes, %Shipments and %Total Cost can exceed 100 percent. This is due to how the formulas are calculated as detailed below.

The Solution Summary screen contains the following information:

  • Solution ID to which the solution summary applies
  • Project ID as specified on the bid round associated with the scenario/solution
  • Bid Round ID as specified on the scenario/solution
  • Total Number of Lanes included on the lane summary criteria associated with the project
  • Total Number of Historical Shipments included in the shipment set associated with the lane summary
  • Domain in which the scenario was created
  • Parameter Set ID as specified on the scenario/solution
  • Rule Set ID as specified on the scenario/solution
  • Shipment Type as specified on the lane summary criteria
  • Awarded Service Providers grid which contains the following:
    • Service Provider ID
    • Lanes - number of lanes awarded to the service provider
    • %Lanes - percentage of lanes awarded to the service provider. This field is calculated as follows: (number of lanes awarded to the service provider/total number of lanes) * 100.
    • Shipments - number of shipments awarded to the service provider
    • %Shipments - percentage of total shipments awarded to the service provider. This field is calculated as follows: (sum of count of shipments awarded to the service provider for this scenario/total number of historical shipments) * 100.
    • Cost - cost of shipments awarded to the service provider
    • %Total Cost - percentage of total costs awarded to the service provider. This field is calculated as follows: (sum of cost for bids awarded to the service provider/sum of cost for bids selected for the solution) * 100.
  • Backup Service Providers grid which contains the following:
    • Service Provider ID
    • Lanes - lanes awarded to this backup service provider
    • % Lanes - percentage of total lanes awarded to this backup service provider

Viewing the Solution Summary

  1. Select an existing scenario.
  2. Click Actions > Solution Summary.

    You see the Solution Summary screen.

Viewing Solution Detail

You can view the solution details:

  1. Click Solution Detail at the bottom of the screen.

Adding More Bids to a Solution

To add bids to a scenario/solution, the bids must:

  • Be in the same bid round as the one specified on the selected scenario.
  • Have a status value of COMMITMENT_COMMITTED.

You can add one or more bids to the solution as follows:

  1. Click Add More Bids at the bottom of the screen.

Note: You can view the Manual Awards Report to see the manual awards for each lane compared to that of demand and the total awarded. If the Fixed option on the solution detail record is selected, then the award is considered manual. To view the report, go to Sourcing > Reports and run the Manual Awards Report. This report will only list the equipment group for lanes with a shipment type of OCEAN.
