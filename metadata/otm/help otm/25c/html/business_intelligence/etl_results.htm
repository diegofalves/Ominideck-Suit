This page is accessed via:

  • Transportation Intelligence > ETL Results
  • Global Trade Intelligence > ETL Results

Use this page to monitor the ETL process via the loading history of the ETL process. Based on your input, this page will display the status of the tables loaded in the current ETL process.

Viewing ETL Results

If you want to view the details of the current ETL process, select the option "In Progress" from the LoadÂ  Status list.

Or, you can use any one of the following fields or just click Search to view details.

  • Enter an ID of the ETL session you want to look up in the Session ID field.
  • Enter the name of the table whose details you want to view in the Table Name field.
  • Enter the date (in the format YYYY-MM-DD) of table loading in the Load Time field.
  • Select one of the following options from the ETL Type list:
    • Standard: Select this option to view details from the current ETL process.
    • Lookback: Select this option to view the details from past ETL processes.
  • Select an option from the Load Status list (Success, Error, In Progress, and Completed) to filter the records based on the loading status of the table.
  • Select an option from the Favorites section to view the records based on data that you have/have not marked as favorite, or to view all the records.
