This project captures the count and cost information for Logistics Network Modeling projects and scenarios.

This project is located in the catalog in Shared Folders/LNM Data Visualisation/LNM - Scenario Graphs

If you are adding this project to an Enhanced Workbench, use the project path of /@Catalog/shared/Custom/LNM Data Visualisation/LNM - Scenario Graphs.
