4. TI: Data Visualization

As explained earlier, the Oracle Analytics Server in the TI solution is comprised of the Oracle Analytics Server suite that hosts the TI specific metadata and the data visualization components.

This chapter describes the details of the TI metadata and the data visualization projects.

Note: While it is possible to create a Data Connection in Oracle Analytics Server, only subject area based connections are supported.

Subject Areas

The Transportation Intelligence, Logistics Machine Learning, Logistics Network Modeling Intelligence, and Transportation Operational Intelligence subject areas are structured into multiple analysis folders one for each OTM business object (order release, shipment, invoice, etc). Each analysis folder then includes one facts folder and multiple dimensions folders. It is important to note that only compatible dimensions (and columns underneath) are available in each analysis folder. All columns under the respective facts and dimensions folders are also functionally grouped under headers (Identifiers, Cost, Distance, Flex fields, etc) for ease of identification and usage.

Data Visualization Projects

Oracle Transportation Management provides some pre-defined data visualization projects in Oracle Analytics Server. These projects are used within the Machine Learning Analytics Workbench, the Modeling Analytics Workbench, and the Business Spotlight. These data visualization projects are:

  • Logistics Machine Learning
  • Logistics Network Modeling
  • Transportation Intelligence
  • Transportation Operational Intelligence
