Tender Dashboard

This dashboard contains reports covering the tender performance information available under respective tabs.

The report for this dashboard is located in the catalog in Shared Folders/FTI Reports/_portal/Tender Dashboard

This dashboard is available as a Preferred Dashboard via User Preferences.

The Tender Dashboard uses the following prompts:

  • Year Month Period Prompt
  • Region Prompt
  • Carrier Prompt
