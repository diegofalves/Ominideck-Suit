Network Dashboard

This Network dashboard is typically meant for network routing analysis and it contains reports covering the key metrics pertinent to network routing.

The report for this dashboard is located in the catalog in Shared Folders/FTI Reports/_portal/Network Dashboard

This dashboard is available as Preferred Dashboard via User Preferences.

The Network Dashboard uses the following prompts:

  • Network Year Month Period Prompt
  • Network Region Prompt
  • Network Top and Bottom Prompt

The Network Dashboard has the following tabs:

Statistics

This report captures the statistical distribution of key network metrics over dimensions.

Distribution

This report section shows as table, the distribution of key network metrics over the chosen dimension for the chosen time period and geography.

Trend Analysis

This report captures the trend over time of the network metrics over the chosen network dimension as follows:

  • Yearly or Monthly trend of
    • Total number of order movements, total number of ship units, total weight, total volume, total allocation cost by

      • Transport mode, service provider, equipment group, path, and edge

Trend Analysis

This report section shows the trend of the chosen metric as a micro-chart (trellis) sliced by the chosen network dimension. The period type of the trend can also be chosen from the drop-down list.

The report for this section is located in the catalog in /Shared Folders/FTI Reports/_portal/Network Dashboard/Network â Trend Report
