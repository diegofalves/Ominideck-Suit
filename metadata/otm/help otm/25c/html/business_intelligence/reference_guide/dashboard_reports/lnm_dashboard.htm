Logistics Network Modeling Dashboard

The LNM Dashboard uses the Logistics Network Modeling Intelligence subject area and allows selection of project GIDs via a prompt and comparison of scenarios under this project via different analyses. This dashboard is available as a preferred dashboard in the OTM/GTM user preferences.

This dashboard report is located in the catalog in Shared Folders/FTI Reports/_portal/Logistics Network Modeling Dashboard
