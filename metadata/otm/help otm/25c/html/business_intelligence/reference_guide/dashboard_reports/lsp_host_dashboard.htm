LSP Host Dashboard

The LSP Host Dashboard contains tabs covering the key business areas under respective tabs for the LSP hosts.

This report is located in the catalog in Shared Folders/FTI Reports/_portal/LSP Host Dashboard

This dashboard is available as Preferred Dashboard via User Preferences.

The LSP Host Dashboard contains the following prompts:

  • Year Month Period Prompt
  • Region Prompt
  • Perspective Prompt
  • Domain Prompt

Below is the list of tabs for the LSP Host Dashboard.

  • Statistics: This tab captures the key statistical details related to shipments for the LSP hosts. This tab contains the same reports present in Statistics tab of Main Dashboard. This tab also contains link to buy versus sell shipments for all statistical details.
  • Financial: This tab captures the key financial details related to shipments for the LSP customers. This tab contains the same reports present in Financial tab of Main Dashboard. This tab also contains link to buy versus sell shipments for all financial details.
  • Utilization: This tab captures the key utilization KPIs of shipments for the LSP hosts. This tab contains the same reports present in Utilization tab of Main Dashboard. This tab also contains link to buy versus sell shipments for all key utilization KPIs of shipments.
  • Carrier: This tab captures the key carrier related KPIs on shipments for the LSP hosts. This tab contains the same reports present in Carrier tab of Main Dashboard. This tab also contains link to buy versus sell shipments for all carrier related KPIs on shipments.
  • Cost Benefits: This tab captures the cost related KPIs for shipments for the LSP hosts. This tab contains the same reports present in Cost Benefits tab of Main Dashboard. This tab also contains link to buy versus sell shipments for all cost related KPIs for shipments.
  • Regional Cost Benefits: This tab captures the shipment cost related KPIs as applicable to specific regions for the LSP hosts. This tab contains the same reports present in Regional Cost Benefits tab of Main Dashboard. This tab also contains link to buy versus sell shipments for all shipment cost related KPIs as applicable to specific regions.
