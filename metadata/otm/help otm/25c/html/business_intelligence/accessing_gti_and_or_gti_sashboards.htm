Accessing Transportation Intelligence and/or Global Trade Intelligence Dashboards

If you have both Transportation Intelligence (TI) and Global Trade Intelligence (GTI) installed, a single Oracle Transportation Management user can only view either TI dashboard, LNM Dashboard, or GTI dashboard. They cannot view more that one dashboard. The following tables show how an Oracle Transportation Management user can be set up to view either the TI dashboard, LNM dashboard, or the GTI dashboard.

Note: To view the TI, LNM, and GTI dashboards, create one user with access to TI and set their dashboard preference to point to the TI dashboard. Then, create a second user with access to GTI and set their dashboard preference to point to the GTI dashboard. For LNM, create a third user Â with access to LNMI and set their dashboard preference to point to the Logistics Network Modeling dashboard.
