Supplier Solicitation: Sample Scenario

You are a manufacturer importing electrical transformers from Canada to Mexico. You have run the optional Tariff Eligibility Screening action and determined that the electrical transformers are eligible to take advantage of USMCA. You want to perform supplier solicitation to solicit supporting qualification data including a certificate of origin. In addition, you want to confirm the country of origin and classification information.

From a configuration perspective, you can create:

    • Create Data Configuration â create a data configuration with an Association Type = Partner Item to Campaign and include all the details you want to copy to the campaign lines upon creation. In this scenario, you want to copy:
      • Item-Campaign Line â item origin details
      • Partner Item-Campaign Line â reference numbers containing the supplier item number
    • Approve Data Configuration â create a data configuration with an Association Type = Campaign Line to Item and include all the details you want to copy to the item-based objects upon approval. In this scenario, you want to copy:
      • Campaign Line-Item â product classification details
      • Campaign Line-Item Origin â item origin details
      • Campaign Line-Item Qualification â item qualification details and standard documents
  • Logic Configuration â create a logic configuration with a Logic Configuration Type = GTM CAMPAIGN CONFIGURATION. In this scenario, you configure:
    • DISPLAY PRODUCT CLASSIFICATION DETAILS â Set Parameter Value to YES
    • DISPLAY QUALIFICATION DETAILS â Set Parameter Value to YES
    • APPROVE DATA CONFIGURATION â Set Parameter Value to the Approve Data Configuration previously created.Â 
    • CREATE DATA CONFIGURATION â Set Parameter Value to the Create Data Configuration previously created.

      • Trading Partner Items - this is your suppliersâ item and includes the Trading Partner Type, Trading Partner, dates and reference numbers/remarks identifying your supplier item.

      • Product Classifications - specify the product classification information for the importing country.

Note: Make sure you have downloaded trade agreements, tariffs and product classification codes from your third-party content provider.

Now you are ready to create your campaign. You, as the campaign administrator, can:

  • Create the campaign or add to an existing campaign from the Item or Trading Partner Item or via Process Management. When you create the campaign, data is copied to the campaign lines based on the details contained in the logic configuration on the campaign type. On the action, you can specify additional details.
  • Once the campaign and associated lines are created, you can send notification, including a document template, to your suppliers. Each campaign line is specific to one supplier/trading partner.Â 

Your suppliers can then add the details to the campaign lines assigned to them.Â 

  • In this scenario, they can add:
    • Product Classification Code for the Product Classification Type defined
    • Country of Origin along with Effective/Expiration Date
    • Qualification details including Qualified Status, Preference Criteria, Producer and Qualification Effective/Expiration Dates
    • Standard Documents and Additional Documents
  • Suppliers can then respond to the campaign administrator and let them know their lines are ready for review. This can be done using the Respond to Campaign Line action.Â 

You, as the campaign administrator, can now review the campaign lines. Once you are satisfied with the response, you can run the Approve Campaign action at either the Campaign or Campaign Line level. This action copies data from the campaign lines to the item, item origin, and item qualification record and anything else you specified in the data configuration with association type = Campaign Line - Item.
