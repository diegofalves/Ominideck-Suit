Item Classifications

This page is accessed via Master Data > Item Classifications.

Use this page to classify items. See About Product Classification for more details.

Header

Classifying an Item:

  1. Enter a product classification type in the Product Classification Type ID field.
  2. Enter a classification code in the Product Classification Code field.
  3. Select the purpose of the classification from theÂ Purpose drop-down list.
  4. Select a domain in which this classification will be active from the Domain Name drop-down list.
  5. Enter an ID of the item that is to be classified in the Item ID field.
  6. Enter a number in the Binding Ruling Number field. Binding rulings could be used to justify a decision to use a particular product classification code for a particular item.
  7. Enter the date from when the binding ruling is effective in the Binding Ruling Effective Date field.
  8. Enter the date till when the binding ruling is effective in the Binding Ruling Expiration Date field.
  9. Enter an ID in the TariffÂ Rules of Interpretation ID field.
  10. Enter any notes related to this classification in the Product Classification Notes field.
  11. Enter any description, if desired, in the Customs Description field.

In View and Edit modes, you will see the Item Classification ID, Item Type, Item Name, Item Description, Product Classification Status, Effective and Expiration Date, and Approver Notes information. The user who initially classified an item and the user who last updated the classification, the related dates, etc. will also be displayed in fields Classification User and Classification Date. The user who reviewed the classification and the date of review will be displayed in the fields Review User and Review Date. The user who performed eligibility screening and the date of screening will be displayed in the fields Eligibility Screening User and Eligibility Screening Date.

Remarks

Use this section to record remarks associated with the item classification.

Adding a Remark:

  1. Select a remark qualifier from the Remark Qualifier ID drop-down list.
  2. Enter your remarks to the Remark Text field.
  3. Click Save.

Reference Numbers

Use this section to record reference numbers that apply to this item classification. You can add as many reference numbers as you require.

Adding a Reference Number:

  1. Select a reference number qualifier from the Reference Number Qualifier ID drop-down list.
  2. Enter the reference number in the Reference Number field.
  3. Click Save.
  4. Click Finished.

Lifetime Events

When the product classification on a trade item is changed, it triggers lifetime events so that the agent actions such as sending notification to a particular contact can be configured. For example, "ITEM CLASSIFICATION - CREATED"agent event with the restriction 'ECCN US' can be configured such that when an ECCN US classification is added to a trade item, a notification should go to the US contact. If ECCN EU classification is added, the EU contact should be notified. Â 

Following are the lifetime/status events for data query type of 'GTM ITEM CLASSIFICATION':

  • ITEM CLASSIFICATION - CREATED: You can select a restriction of Product Classification Type.
  • ITEM CLASSIFICATION - MODIFIED: You can select a restriction of Product Classification Type.
  • ITEM CLASSIFICATION - STATUS CHANGE: You can select a restriction of Product Classification Type and/or Classification status (CREATED,APPROVED,DECLINED,INVALID).
  • ITEM CLASSIFICATION - REMOVED: You can select a restriction of Product Classification Type.
