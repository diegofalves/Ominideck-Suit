Campaign Lines Document Details

This page is accessed via Trade Agreements > Campaign Lines. Then click the Documents tab. Click the Details icon ().

Use this tab to manage documents associated with the campaign line.

Revisions

In this section, revisions to the selected document are listed.

Use the following icons to interact with standard document revisions.

  • Actions (): Launches the Campaign Lines Document Revision Actions page.
  • View (): Views the content of the latest document revision in a popup.
  • Details (): Views the details of the standard document revision.

Status

This section lists status types and status values. This information enables you to track the progress of the document filing.

Text Templates

View any text template associated with the document. A standard document can be associated with one or more text templates, based on the document type and campaign line. For example, a HAZMAT warning can be necessary on a bill of lading if the shipment holds hazardous material.

Reviewers

This section displays any reviewers assigned to this document. Reviewers receive notifications of document state and content changes. Each reviewerâs approval is required before the document content can be finalized.

Subscribers

This section displays any subscribers assigned to this document. Subscribers receive finalized content.
