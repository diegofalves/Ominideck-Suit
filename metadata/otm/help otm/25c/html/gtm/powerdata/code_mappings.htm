Code Mappings

This page is accessed via Master Data > Power Data > Configurations > Code Mappings.

Use this page to add/modify/delete code mappings based on the latest customs data changes. For example, you can map license code and license exception code for Automated Export System (AES) report generation.

Adding/Modifying/Deleting a Code Mapping

  1. Enter a unique identifier in the Code Mapping ID field. You will use this ID when you refer to this code mapping on other pages.
  2. Select the required option from the the Code MappingÂ Type drop-down list.
  3. Click Finished.
    This will create a new code mapping ID and copy over the pre-defined mappings (Authorization, Exception Control Code, License Code, and Authorization Type) that are specified by the parameter P_CUSTOM_MAPPING_ID.
  4. To add a mapping, click Add Code Mapping in any section.
    1. Enter a new code in the Code field.
    2. Enter the code that is to be overridden in the Override External Code field.
    3. Click Save.
  5. To edit any mapping, click the Edit icon corresponding to the required record.
    1. Enter the code that is to be overridden with in the Override External Code field.
    2. Click Save.
  6. Click the Delete iconÂ corresponding to the required record to remove an updated/newly created record.

Copying a Code Mapping

On the results page, select a record and click Copy Code Mapping to create a new code mapping ID with the default settings.
