Trade Incentive Program Rules

This page is accessed via Master Data > Power Data > Trade Incentive Programs > Trade Incentive Program Rules.

Use this page to create rules to determine trade incentive programs that are applicable for different kinds of products.

Adding a Trade Incentive Program Rule

  1. Enter a unique identifier in the Trade Incentive Program Rule ID field.Â You will use this ID when you refer to this rule on other pages.
  2. Enter the name of the trade incentive program rule in the Name field.
  3. Enter a classification type ID in the Product Classification Type field.
  4. Select a domain in which this trade incentive program rule will be active from the Domain Name drop-down list.
  5. If desired, enter additional information about theÂ trade incentive program rule in the Description field.
  6. Select the Active checkbox if theÂ trade incentive program rule is to be active.
  7. Enter dates in the Effective Date and Expiration Date fields which define the start and the end date of the trade incentive program rule.
  8. Enter an ID in the Trade Incentive Program field.
  9. Enter a sub type in theÂ Trade Incentive Program Sub Type field.
  10. Enter the requiredÂ saved conditionÂ in theÂ Exit Saved Condition field. The applicable trade incentive program rule for an exit line is determined based on this query when you run the actionÂ Perform Entry/Exit Recording.
  11. Select the required option from the Accounting Method drop-down list.
  12. Enter the number of years, months, and days in the Period of Discharge fields.
  13. Enter a constraint set ID in the Matching Constraint Set field to specify the constraints configured for this rule.

Product Specific Rules

Use this section to associate a specific range of product classification codes with the trade incentive program rule.

  1. Enter product classification codes in the Start Classification Code and the End Classification Code fields. The rules will be applied to the products having the codes mentioned in these fields.
  2. Enter required numbers in theÂ Period of Discharge - Years, Period of Discharge - MonthsÂ and Period of Discharge - Days fields.
  3. Enter a constraint set ID in the Matching Constraint Set field.
  4. Click Save.

Reference Numbers

Use this section to record reference numbers that apply to this trade incentive program rule.Â 

Adding a Reference Number:

  1. Select a reference number qualifier from the Reference Number Qualifier ID drop-down list.
  2. Enter the reference number in the Reference Number field.
  3. Click Save.

Remarks

Use this section to add as many remarks as you require.

Adding a Remark:

  1. Select a remark qualifier from the Remark Qualifier ID drop-down list.
  2. Enter your remarks in the Remark Text field.
  3. Click Save.

Text Templates

Use this section to specify templates that will be used with this trade incentive program rule.

Adding a Text Template:

  1. Enter an ID in the Text Template ID field.
  2. Enter a type in the Document Type field.
  3. Enter any related text in the Text field.
  4. Click Save.
  5. Click Finished.
