Parties

This page is accessed via:

  • Restricted Party Screening > Parties
  • Master Data > Parties

Use this page to view existing parties and to record party information.

Identification

Adding a Party:

  1. Enter an ID for the party in the Party ID field.
  2. If the party is an individual, use the First Name and Last Name fields to record the full name.
  3. Enter an alternate name in the Alternate Name field.

    Note: GTM bases restricted party screening on the party's company name. If the contact has an alternate name, the contact's alternate name is screened instead.

  4. If applicable, enter a name in the Company Name field.
  5. Enter an ID in the Location ID field.
  6. Select a language from the Language Spoken drop-down list.
  7. If applicable, enter a title in the Job Title field.
  8. Use the Email Address, Fax, Phone 1, Phone 2, and Cell Phone fields to record additional party information.
  9. Enter any descriptive information in the Description field.
  10. Select a domain from the Domain Name drop-down list.
  11. If you are looking at an existing record, click Status to see the internal statuses assigned to the record.

Address

This section displays address information associated with the Location ID you selected in the Identification section. If you did not designate a Location ID, no address information will be displayed.

Note: This section will only be visible if you are viewing or editing an existing record. It will not be visible if you are creating a new record.

User-defined Code

  1. Select a user-defined code category from the Category drop-down list.
  2. Select a type ID from the Type ID drop-down list.

    Note: The available options are determined by your choice of category in the previous step.

  3. Select a code type from the Type Code drop-down list.

    Note: The available options are determined by your choice of type in the previous step. For easy understanding of the user, this drop-down list displays description of the user-defined code values instead of only the codes.

  4. Click Save.

Registration

Use this section to record registration information if it is available to you.

  1. Select a registration ID from the Registration ID drop-down list.
  2. Select a qualifier from the Involved Party Qualifier ID drop-down list.
  3. Click Save.

Reference Numbers

Use the reference numbers section to add numerical information you wish to associate with this party.

  1. Select a reference number qualifier ID from the Reference Number Qualifier ID drop-down list.
  2. Enter a reference number in the Reference Number field.
  3. Click Save.

Remarks

Use the remarks section to add text you wish to associate with this party.

  1. Select a remark qualifier ID from the Remark Qualifier ID drop-down list.
  2. Enter the remark in the Remark Text field.
  3. Click Save.
  4. After filling out this page, click Party Screening Matches to continue, or Finished if you are finished entering contact information.
