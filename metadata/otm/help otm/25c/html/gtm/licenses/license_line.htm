Licenses and Registrations

License Lines

This page is accessed via:

  • Licenses and Registrations > License Lines
  • Licenses and Registrations > Licenses. Click the License Lines tab and then click New License Line.

Use this page to add new line items to a license.

Identification

  1. The License Line ID and the License ID associated with the license line are displayed.
  2. Enter the commodity for which the license line is used in the Commodity ID Â field.
  3. Enter a commodity ID in the User Defined Commodity ID field.
  4. Select an option from the Line Type drop-down list. Your selection will determine whether the license line will be used for used for import or export purposes.
  5. If desired, enter a description for the license line in the License Line Description field.
  6. Enter a formula in the Product Classification Matching Formula field.
    OR
    Enter an ID in the Product Classification Group ID field.
    When "License Line Product Classifications" is a constraint configured for determining a license for a transaction line:
    a. if a product classification group is specified on a license line, a transaction line classification code should also be present in the product classification group to be considered a match.
    b. if a product classification matching formula is specified on a license line, it should match with the classification code on the transaction line. Otherwise, it will not be considered a match.
    c. if neither a product classification matching formula nor a product classification group is specified on a license line, the application will consider it as a match and evaluate other constraints.

Items

  1. Enter an ID for the line item in the Item ID field.
  2. Click Save.

Quantity and Value Balances

In this section you can modify the authorized quantity and value approved for the license line, and view information about values and quantities which remain available on the license.

  1. Enter a quantity in the Authorized Quantity field.
  2. Select a unit of measure from the UOM drop-down list.
    Note: You can configure the values displayed in this drop-down list via the property gtm.licenseLine.displayUOMTypesForQuantity.
  3. Enter a value in the Authorized Value field, and select a currency from the corresponding drop-down list.

    Note: If a license is already assigned to a license line, the Authorized Quantity and Value fields are displayed as read-only fields. You can view information but you cannot modify the values in these fields.

You can view the following information:

  • Used Quantity and Value: These fields indicate the quantity and value of the license assigned for any trade transaction line (shipment line).
  • Reserved Quantity and Value: These fields indicate the quantity and value of the license reserved for a sales order or a fulfillment line.
  • Available Quantity and Value: These fields indicate the quantity and value of the license that can be used to make reservations for new sales orders.

Values

Use this section to add value-related information to the license line.

  1. Select a qualifier from the Value Qualifier ID drop-down list.
  2. Enter a value in the Value field, and select a currency from the corresponding drop-down list.
  3. Click Save.

Remarks

Use this section to add as many remarks as you require.

  1. Select a qualifier from the Remark Qualifier ID drop-down list.
  2. Enter the remark text in the Remark Text field.
  3. Click Save.

Reference Numbers

Use this section to add numerical information you wish to associate with this license line.

  1. Select a reference number qualifier from the Reference Number Qualifier ID drop-down list.
  2. Enter the reference number in the Reference Number field.
  3. Click Save.

Transaction Line Saved Query

Use this section to add one or more saved queries. Licenses will be evaluated and assigned to a trade transaction line based on these queries.Â 

  1. Enter an ID in the Saved Query field. A license will be matched based on this query during license determination via constraint set matching.
  2. Click Save.
  3. Click Finished.
