Licenses and Registrations

Licenses

This page is accessed via Licenses and Registrations > Licenses.

Use this page to create, view, and edit import and export licenses.

Note: If you accessed this page via the Copy License action, see Copy License for more information.

Note: In GTM for screening purposes, licenses take precedence over controls.

Note: License Management is intended to support static licenses. It is not intended to support dynamic creation of licenses based on transactions and/or declarations.

Identification

  1. Enter an ID for the license in the License ID field.
  2. Enter a value in the License Number field.
  3. If you are looking at an existing license, click Status to open a window which displays any external status which is assigned to the license.
  4. If this license is to be active, select the Active option.
  5. If this license is a template, select the Template option.
  6. Select the domain in which this license will be active from the Domain Name drop-down list.
  7. Select a category from the License Category drop-down list.
  8. Select a type from the License Type drop-down list.
  9. Select a code from the License Code drop-down list.
  10. Enter jurisdiction and regime information in the Jurisdiction and Regime fields.
  11. Use the date fields to record date information about the license.

    Note: Licenses with past expiration dates will be picked up if the transaction date falls between the license effective date and expiration date.

Control Type Code

If this license is associated with one or more specific control codes, use this section to define the relationships.

  1. Select a control type from the Control Type drop-down list.
  2. Select the appropriate control code from the Control Code drop-down list.
  3. Click Save.

Date

Dates recorded in this section are for informational purposes only and are not referred to by the software when screening occurs.

  1. Select a qualifier from the Date Qualifier ID drop-down list. This list will be populated with date qualifiers which you have set up.
  2. Enter a value in the Date field.
  3. Click Save.

Ports

  1. Select a port qualifier from the Port Qualifier ID drop-down list.
  2. Enter the port ID in the Port ID field
  3. Click Save.

Region

Use this section to define regions that are involved in this license.

  1. Select a region qualifier from the Region/Country Qualifier ID drop-down list.

    Note: Options on this drop-down list are derived from the involved party qualifiers.

  2. Enter a region ID in the Region ID field.
  3. Entry the appropriate country code in the Country Code ID field.
  4. Click Save.

Involved Parties

You can associate a wide variety of involved parties to a license.

  1. Select a qualifier from the Involved Party Qualifier ID drop-down list.
  2. Enter a contact in the Involved Party Contact field.
  3. Click Save.

Related Licenses

  1. Enter the ID of a related license in the License ID field.
  2. Click Save.

Related Registrations

  1. Enter the ID of any related registrations in the Registration ID field.
  2. Click Save.

Reference Numbers

Use the reference numbers section to add numerical information you wish to associate with this license.

  1. Select a reference number qualifier ID from the Reference Number Qualifier ID drop-down list.
  2. Enter a reference number in the Reference Number field.
  3. Click Save.

User-defined Code

You can associate user-defined codes with a license.

  1. Select a code category from the Category drop-down list.
  2. Select a code type from the Type ID drop-down list.

    Note: The available options are determined by your choice of category in the previous step.

  3. Select a code value from the Type Code drop-down list.

    Note: The available options are determined by your choice of type in the previous step. For easy understanding, this drop-down list displays description of the user-defined code values instead ofÂ codes.

  4. Click Save.
