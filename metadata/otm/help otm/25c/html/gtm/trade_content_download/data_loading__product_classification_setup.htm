Data Loading: Product Cassification Setup

For classification data there are a few additional areas to be aware of including Product Classification Category and Product Classification Type. Product Classification Code is discussed in the Data Download topic.

Product Classification Category

The product classification category defines the high level details for product classification data. Oracle Global Trade Management (GTM) ships with pre-seeded product classification categories including ECCN, HS, ML, and Schedule B. Currently, these are the only product classification categories supported by the GTM content loader. GTM and the content provider may be able to support other product classification categories but work is required by Oracle and the content provider during implementation.

Product Classification Type

The product classification type defines product classification data by region. It is specific to a product classification category and a region or a country.

The recommended naming convention for product classification type is [type of classification data][space][2-character country code/region code]. If there is a separate export tariff, the recommended naming convention is [type of classification data]-EX[space][2-character country code/region code]. A few examples are:

  • ECCN US â Export Control Classification Number of the United States
  • HTS EU â Harmonized Tariff Schedule of the European Union
  • ML MY â Munitions List of Malaysia
  • HS-EX DE â Harmonized Tariff Schedule of Germany (Export)
  • HS-X4 JP â Harmonized Tariff Schedule of Japan (10-digit list)

For example, you can add a product classification type if you want to receive the harmonized tariff for Brazil.

Navigate to Master Data > Power Data > Product Classification > Product Classification Types.

  • Enter a Product Classification Type = HTS BR.
  • Select a Product Classification Category = HS.
  • Enter a Description = Harmonized Tariff Schedule for Brazil.
  • Select a Country Code = BRA.
  • Select a Content Type ID = HTS-BR (this is the Content Type created previously).
