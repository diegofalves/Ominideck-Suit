Propagate Changes to Declarations

This page is accessed via:

  • Transactions > Trade Transactions > Actions > Manage Declarations > Propagate Changes to Declarations
  • Transactions > Trade Transaction Lines > Actions > Manage Declarations > Propagate Changes to Declarations
  • Customs > Declarations > Actions > Manage Declarations > Propagate Changes to Declarations
  • Customs > Declaration Lines > Actions > Manage Declarations > Propagate Changes to Declarations

Note: This action cannot be run on a declaration created using the old Create Customs Shipment action.

Data (other than quantities) on trade transaction lines can be modified even after declarations are created. Use this action to sync the declaration lines with any such modified transaction line data. At the same time, all the declaration lines will be re-evaluated against declaration configuration logic of transaction line grouping, aggregation, and data copy. If required, transaction lines will be re-aggregated according to the selected logic criteria to modify the existing declaration lines.

If a trade transaction line is removed after a declaration is created from it, this action will remove the trade transaction line from the declaration.

Note: If you want to modify quantities of transaction lines after a declaration is created, use the Edit Declaration Quantities action to update the declaration.

No input is required to run this action.

After you have performed this action, the Results page will display the list of the declarations impacted.
