Use this action to add the selected item/partner item to an existing campaign.

Adding an Item to a Campaign

  1. Enter an ID of the campaign to which you want to add this item in the Campaign ID field.
  2. Select one or multiple documents from theÂ Required Documents drop-down list. When this action is run, the selected documents will be attached with the campaign.
  3. Click Add to Campaign. This action creates a campaign line for each partner/site associated with the item/partner item, copying over pertinent information from the selected campaign such as Required Documents, etc.
