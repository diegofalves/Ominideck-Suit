Estimate Duty and Taxes

This page is accessed via:

This action helps you to send trade transaction information or declaration information to a third party data provider and obtain duty and tax values for a trade transaction, trade transaction line, declaration, or a declaration line.

When you select a record and execute this action, information like shipment ID, Incoterm, transport mode, line IDs, classification code imported for each line, value for individual lines, and quantity of goods is sent to the third party data provider. The duty and tax details obtained from this data provider for any line are saved against that line along with formula and notes. The taxes obtained for the trade transaction are saved against that trade transaction.

Note: You can perform this action on a maximum of 10 records at a time.

The data provider reverts with a breakup of the duty and taxes (base duty and details of duty types like penalty, exemption, etc.) for the product classification codes. This information is displayed if you click the Details link displayed against the value qualifier "DUTY" in the Values section of a trade transaction lineÂ or a declaration line.
