Classify Item

This page is accessed via:

  • Classification > Items > Actions > Classify Item
  • Master Data > Items > Actions > Classify Item

Use this page to look up existing product classifications.

Note: You cannot define new product classification codes on this page. Â For help with creating new codes, see Product Classification Codes.

  1. Select a type from the Product Classification Type drop-down list. Product classification types are defined in Power Data. See Product Classification Type.
  2. Enter a classification code in the Product Classification Code field. Â 
  3. Enter the product description in the Description field.

    Note: If you specify both the classification code and the product description, the search will ignore the description and conduct the search based on the classification code.

    Note: You must select either a classification code or enter a description in addition to selecting a product classification type in order for the lookup to produce results.

  4. Use the default language displayed or select a language from the Language drop-down list. Also see Logic Configuration - GTM Product Classification UI Configuration.
  5. Use the Transaction ID field to record an existing transaction ID if it applies to the product lookup.
  6. Specify a source system in the Source System field if it applies to this lookup.
  7. Select a service preference from the Service Preference ID drop-down list. Â A service preference determines the granularity level of the search results.

    Note: You can define a new service preference. See Service Preference.

  8. If you wish to view the results in an expanded tree, select the Tree Results checkbox.
  9. Click Match. Your matches are displayed. They are arranged in hierarchal order. See Select Product Classification for more information.

    Note: To see any duties or taxes associated with an item, click the product classification code on the Classification window and then click View Duties and Taxes. Global Trade Management will contact the third party data provider to obtain the information which will then be displayed to you.

  10. To see a list of your search criteria, click Search Criteria.
  11. To run another match, click Match Again.
  12. To refine your search criteria, click Refine Match.
  13. To re-run the match, click Rerun Match.
