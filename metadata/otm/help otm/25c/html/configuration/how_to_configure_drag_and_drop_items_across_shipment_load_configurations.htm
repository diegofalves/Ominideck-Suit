Single or multiple items movement across shipment load configuration

Let's assume that there are 3 regions having 3D load configurations in the workbench layout. They are: Load Configuration 1 and Load Configuration 2, and Load Configuration 3.

To move a single or multiple items from Load Configuration 1 to either Load Configuration 2 or Load Configuration 3,Â  follow the below steps

  1. Select a single item to move from a load configuration to another, mouse over the item and RIGHT CLICK to select it. Once selected, the item color will change toÂ grey.
  2. Select multiple items, press and hold the CONTROL key, and LEFT CLICK to select an item. Then, RIGHT CLICKÂ on the item showingÂ Move to Shipment. OnceÂ selected, the items will change toÂ grey.Â Â 
  3. Click on the Move to Shipment pop up message to move the chosen items from Load Configuration 1 to Load Configuration 2 or Load Configuration 3.
  4. Select the target Load Configuration and DOUBLE CLICK inside the equipment wire frame. The Select Ship Unit Pieces pop up screen will display the items that will be moved.
  5. Remove the check mark from any items that should not be moved. Click Done.
  6. The load configuration will automatically go into edit mode and the items will be placed in the parking lot. Click Save.
  7. Moved items are added to the target shipment. The target shipment gets repacked and its 3D load configuration is reloaded.
  8. Edit the newly added items if the current placement is not optimal.
