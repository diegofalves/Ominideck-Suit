About Grouping Location Resources

This topic describes how to set up a location with location resources (dock doors) and location resource groups along with appointment rule sets.

Follow these steps to group location resources (dock doors)

  1. Create a location. Use Location Manager to record and store information about locations.
  2. Select your Location Resource Group. Â Now, assign the location which you have created to this location resource group. Â Also, you can define default or seasonal appointment rule sets.
  3. Go to the location which you have created and then click the Resource tab. Under the Location Resource Group section, you can view all the location resource groups assigned to this location.
  4. On the Resource Type grid, you can either modify or create a new Location Resource Type. Â On the Location Resource Type page, you can define a calendar and manage location resources. If you select the Constrain Appointment checkbox, the appointment rule sets defined on the location resources or location resource groups will be considered for resource compatibility check, else appointment rule sets are ignored.
  5. Under the Location Resource section, click New Resource or edit an existing resource. On the Location Resource page, select the appropriate location resource group or define appointment rule sets.

    Note: You can either have appointment rule sets on the location resource directly or you can select a location resource group available for this location and define the appointment rule sets on that group. Â 

    • If the location resource group is not selected, the location resource remains ungrouped.
    • Appointment Rule Sets enable you to add various constraints along with effective and expiration dates.
    • When choosing which calendar to use, OTM uses the following hierarchy:
      • location resource
      • location resource group
      • location resource type
      • If no calendar is defined on any the above, OTM displays an error message. Â 
  6. When the location setup is ready, you can Manage Appointments or Schedule Appointments.
    • When you run the Manage Appointment action, you can view various location resource groups with multiple location resources across location resource types. Â 
    • When you run the Schedule Appointment action, you can either select an appointment from the list of location resource group options or you can click View Group Resources, to select a specific resource of the selected location resource group. Â
