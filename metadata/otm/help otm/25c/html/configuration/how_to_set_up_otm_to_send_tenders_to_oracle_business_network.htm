The Oracle Business Network (OBN) is a secure cloud-based Business-to-Business (B2B) collaboration platform that automates electronic transactions between Oracle application customers and their trading partners. To send tenders to OBN, set up an external system with the OBN URL and details.Â 

Setting up Service Provider and External System to Send Tenders to OBN

  1. Set up the external system for the external rating engine using REST Client to connect via OBN.
  2. Set up the external system for tender using the following configuration details:
    Create anÂ External System with the following values underÂ Business Process Automation > Communication Management > External Systems

    • Authorization Service URL:Â https://businessnetwork-stage.oracle.com/restservice/v2/token
    • URL:Â https://businessnetwork-stage.oracle.com/restservice/receiver
  3. On the Service Provider Manager, add Reference Number Qualifiers with Reference Number Qualifier IDs as OBN_RECEIVED_ID_TYPE and OBN_RECEIVED_ID. (The values hold the unique ID and the type of the ID given to the trading partner when registered with OBN). You can create and add your Out XML profile in the Out XML Profile ID field.

    Note: Out XML Profile ID âGLOG_OBN_TENDEROFFERâ is staged for your reference.

  4. Choose the communication method (e.g. HTTPPOST or REST) under the Communication and Remarks tab of Service Provider Manager. Select the external system that you created in step 1 to send the tenders to OBN.

Note: To perform these actions, you need to have the following User Manager access permissions:
-> REST - Tender Actions
-> REST - Tender - Update
-> REST - Tender - View

When you send the tender, the system checks the communication method, and TenderOffer XML will be sent to the external system that is configured to this communication method.

Note: While awarding the tender to a digital freight carrier, if there is an external quote ID returned by this carrier, then the value of the EXTERNAL_QUOTE_ID will be copied to the shipment reference number EXTERNAL_QUOTE_ID.
