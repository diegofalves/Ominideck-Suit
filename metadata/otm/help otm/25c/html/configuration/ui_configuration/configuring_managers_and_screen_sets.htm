Configuration and Administration

Configuring Managers and Screen Sets

Many managers can be configured. There are two main portions to configure: screen sets, which control search and search results pages; and manager layouts, which control the new, edit, and view pages for a manager. You can configure either or both.

  1. Create a screen set. It is recommended that you copy an existing screen set and modify it.
  2. Create a manager. It is recommended that you copy an existing manager layout and modify it.
  3. Update the screen set so that on the General tab, you use the new manager that you just created. Separate managers may be created for new, edit, and view modes.
  4. Add the screen set to a new menu.

Configuring Actions

Some actions may be configured as part of a manager layout.
