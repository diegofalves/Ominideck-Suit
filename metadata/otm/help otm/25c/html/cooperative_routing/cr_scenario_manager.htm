Cooperative Routing

Scenario

This page is accessed via Cooperative Routing > Route Analysis > Scenario.

The cooperative routing scenario defines the parameters and inputs for a cooperative routing solver including the constraints, and the fleets and related fleet counts. These include a scenario constraints (to be used for fleets that do not have their own constraints defined), a list of fleets to be considered in the solve and the associated fleet resource counts. It also includes parameters for the cooperative routing solver in order to control solution quality and run time.

Adding a Scenario

  1. Enter a unique code for the scenario in the Scenario ID field.
  2. Enter the project to which this scenario belongs in the Project ID field.
  3. Select the Domain Name for the scenario.
  4. In the Max Solver Runtime field, enter the maximum time that you want the solver to run.
  5. Optionally, enter the Improvement Cutoff Percent to define the minimum improvement between iterations upon which the solver will proceed to the next iteration. This is a run-time control that stops the solver when the solution is not improving much.

    Note: If both the max solver run-time and the improvement cutoff are blank, the solver will run until it can no longer find any improvement.

  6. Optionally, in the Constraint Set ID field, enter default constraints for the scenario. These constraints will be used if the corresponding constraints in a fleets constraints are empty.
  7. In the Min Intra-Region Distance field, enter the distance to use for deadhead (empty transport) distances within the same region. This is not required, however if not defined, a deadhead distance within the same region will be whatever the distance engine produces for a distance between the region's representative location and itself. This is meant to be a minimum distance value used within the same region, not an absolute override for the calculated rate distance.
  8. If you want, enter a Description in the corresponding field.

Fleet

  1. Enter the Fleet ID.
  2. Enter the Num Resources.
  3. Click Save after adding each fleet.
  4. Click Finished.
