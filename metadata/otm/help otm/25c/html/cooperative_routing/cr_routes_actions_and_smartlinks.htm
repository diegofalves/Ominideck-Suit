Cooperative Routing

Routes Actions and SmartLinks

This page is accessed via Cooperative Routing > Route Analysis > Routes. Click Actions at the top of the results page to launch the following actions.

Action

Description

Commit CR Route

Commits the CR route(s), and updates the related data: the committed confidence factors of all CR routes which share a lane volume forecast with this route and the lane commitment data for all the lane volume forecasts within this route.

Convert to Route Template

This action takes a cooperative routing route and converts it to a route template.

Copy CR Route

This action copies the route.

Map CR Route

Shows cooperative routing (CR) routes and legs on Oracle Transportation Management map.

Process CR Route

If you manually edit a route or create a new one, you can save it, but for the system to compute the distances, time, and cost, you must run this action to process it.

Uncommit CR Route

Uncommits the CR route(s), and updates the related data: the committed confidence factors of all CR routes which share a lane volume forecast with this route and the lane commitment data for all the lane volume forecasts within this route.

SmartLinks

Right-click on any ID to launch the following SmartLinks.

SmartLink

Summary

View Related Lane Commitments

Finds the lane commitments associated with this CR route.

View Similar Routes

This link displays similar routes. One route is similar to another if they visit the same regions in the same order.

Similar routes may differ from one another with regard to the days on which they visit those regions, as well as the time of day, costs, transit times, fleets, volumes, and start and end depots.
