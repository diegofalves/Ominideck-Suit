OracleSpatial Properties

This table lists the properties that OracleSpatial external distance engine can use.

Note: You may use the following to configure Oracle Maps Cloud Service (formerly known as eLocation). However, Oracle Fleet Management customers who want to use the service are required to subscribe to both Oracle Maps Cloud Service Asset Tracking and Oracle Maps Cloud Service Asset Optimization.

Property

New in Version

Description
OracleSpatial.host 6.3.1

This property is used to set the value of the EDE parameter HOST for OracleSpatial distance engine.

The value of the property is elocation.oracle.com
OracleSpatial.port 6.5.1

This property is used to set the EDE parameter PORT value for the Oracle Spatial Distance engine.

By default, this property is set to 443 as the default protocol used will be HTTPS.

6.3.1
