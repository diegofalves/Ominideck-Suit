Agent Actions by Object

The links below take you to tables that list the agent actions for each agent type or business object. Many actions require additional parameters to direct their execution. An agent, for example, can Withdraw Tender and notify the service provider of the withdrawal, or assume the service provider has already been notified via telephone.

Utility actions are general purpose actions that are available to all (or most) business objects. They include support for notification and control flow. Some agent types only use utility agent actions.
