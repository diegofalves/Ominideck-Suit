About Logistics Machine Learning Intelligence

The Logistics Machine Learning subject area provides an environment for reviewing historical shipment data by monitoring, analyzing, and reporting metrics. To provide access to Logistics Machine Learning Intelligence, you need to add a business intelligence application and a business intelligence role to each user as appropriate to access different functionality. Logistics Machine Learning Intelligence is based on Oracle Analytics Server (OAS).

Logistics Network Modeling Intelligence is available as a subject area under Transportation Intelligence > Ad Hoc Query > Analysis > Logistics Machine Learning > Machine Learning.

There is also a pre-defined Logistics Machine Learning Intelligence dashboard available in OAS under Dashboards > LML Dashboards > Logistic Machine Learning Dashboard.

The Logistics Machine Learning subject area consists of machine learning specific data and links to data from Oracle Transportation Intelligence. The machine learning specific data is in the following folders:

  • Machine Learning Shipment Facts
  • Machine Learning Scenario Level Training Analysis
  • Machine Learning Feature Contribution Analysis
  • Machine Learning Shipment Level Training Analysis
  • Machine Learning Prediction Analysis
  • Machine Learning Event Analysis
  • Machine Learning Event Detail Analysis
  • Machine Learning Event Training Results Analysis

Loading Data

To load data to Logistics Machine Learning, use the actions Load Data into Analytics and Load Shipment Data into Analytics.

To ensure the remaining historical shipment related data is available in the Logistics Machine Learning intelligence subject area, run the Transportation Intelligence ETL. To load the ETL, use the Run ETL process.

Dashboard

A dashboard is a collection of performance metrics reflecting common strategic goals. The metrics are displayed as various key performance indicators formatted as charts, graphs, tables, cross-tabs, and status indicators. There are multiple tabs organizing the metrics. Dashboard comes pre-defined with standard metrics.

Ad Hoc Query

This component allows creation of reports on the fly. For details on Ad Hoc Queries, click the OAS help icon. This page is accessed via Transportation Intelligence > Ad Hoc Query. Then under Analysis and Interactive Reporting select Analysis > Logistics Machine Learning > Machine Learning. You should have an OAS license to work in this page.
