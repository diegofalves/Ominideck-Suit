Settings and Actions User Access

This page is accessed via Configuration and Administration > User Configuration > Manage User Access. In the User Access Type field select the option Settings and Actions. Click Edit User Access. The Settings and Actions page opens.

You can configure Oracle Transportation Management so that when a specific user opens the Settings and Actions page certain sections are hidden.

You can control which sections a specific users sees as follows:

  1. Select the Prevent Access Changes checkbox to specify the final level for access configuration.
  2. Select a section of the Settings and Actions page to hide. The options are System Details, Troubleshooting, or Administration.
  3. Click Save for each section you want to hide.
  4. Click Finished.
