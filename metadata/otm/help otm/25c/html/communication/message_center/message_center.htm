Message Center/Notifications

The (Notifications) is a centralized message board that displays messages when specific events occur such as when business objects are created or modified, or status changes. The message center is separate from the Message Hub, which is used to track XML messages between the system and a device or driver. Some messages in the Message Hub may also be configured to appear in the Message Center.

You can also configure Oracle Transportation Management to email or fax contacts when these events occur.

The Notifications icon appears at the top of the main Oracle Transportation Management page. If you have new unread messages, a red circle appears on the icon (). When the mouse pointer hovers over the icon, a tool tip displays the number new messages (# New Notifications). If you click the icon, you see a finder results page in a new window or tab (based on browser settings) which displays all of your messages. To control how often the number of new messages is automatically refreshed, set the glog.ui.messageCenter.refresh property which sets the refresh rate of the Message Center (in seconds).

Note: The counters only count messages sent directly to you. Oracle Transportation Management does not count messages you may have access to read in other domains. This is the reason you may see a different number of messages in the counters compared to when you search for messages.
