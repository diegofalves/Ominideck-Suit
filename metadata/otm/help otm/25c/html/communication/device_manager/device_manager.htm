Device Manager

This page is accessed via Business Process Automation > Communication Management > Devices.

A device represents a mobile communication device or sensor. One example of such a device is something attached to a truck that can wirelessly communicate information about the truck, driver, and its shipments.

Adding a Device

  1. Enter a Device ID.
  2. Enter a Description.
  3. Enter a Contact ID.
  4. Enter a Communication Method ID.
  5. Enter an External Device ID.

Current Device Associations

  1. Enter a Start Date.
  2. Enter an End Date.
  3. Select a Data Query Type ID.
  4. Enter an Object ID. The object ID must be selected after the data query type.
  5. Click Save for each device association.

Reference Numbers

Enter reference numbers.
