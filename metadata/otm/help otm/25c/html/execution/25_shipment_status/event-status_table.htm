Shipment Event - Shipment Status Table

This table summarizes what internal events are raised and what shipment statuses are set out-of-the-box depending on what shipment event group your shipment event triggers. To learn more about other conditions for setting statuses and internal events see the SHIPMENT EVENT TRACKER agent.

Note: The attributes listed are calculated based on other information for the shipment and not defined by you in the shipment event group. See the RECALC ESTIMATED STOP TIMES agent action to learn how these attributes are calculated.

Note: Your site may have changed the SHIPMENT EVENT TRACKER agent to raise and set totally different events and statuses.
