Shipment Management

Distance and Time

This page is accessed via:

  • Ask Oracle > Distance/Time
  • Shipment Management > Ask Oracle > Distance/Time

The Distance and Time query provides the distance and travel time between two geographic points, based on Rate Distance and Rate Service Type.

Enter as much information as possible in the search fields. The more information you enter, the faster and more accurate the search results.

The following properties may be relevant. Each can be set multiple times with different values:

  • glog.askOTM.distanceTime.showRouteCode.extEngineJavaClass: Specifies the java class for an external distance engine that supports Route Code as an input from this page.
  • glog.askOTM.distanceTime.showSCAC.extEngineJavaClass: Specifies the java class for an external distance engine that supports SCAC code as an input from this page.

Ask Oracle About Distance And Time

  1. Choose a Request Type to limit the search query to Distance, Time, or Both. If you select DISTANCE, your query finds the distance for the geographic pair using the Rate Distance ID. TIME finds the time between the geographic points using the Rate Service ID. BOTH finds distance and time information.
  2. Enter a Rate Service ID in the Rate Service text box.
  3. Select a Rate Distance.
  4. If you have PC*MILER Rail installed and have selected it as your rate service type, enter the rail route code in the Route Code field.
  5. Complete the Ship From and Ship To fields. Choose a specific location in Oracle Transportation Management or specify at least one of the following
    • SPLC
    • Rail Station Code
    • City
    • State/Province Code
    • Country
    • Postal Code
    • County Qualifier
    • Latitude and Longitude
    • Zone(s)

    Note: Latitude and Longitude are used by the Oracle Map Cloud Services engine.

    Note: If you are using PC*MILER Streets or PC*MILER Rail, you must define both the source_address_type parameter and the dest_address_type parameter as street_address. The street addresses - including city, state, and ZIP - must then be defined in a source and destination location objects, not the Ask Oracle interface

  6. Limit your search for distance/time by specifying a Ship Date and a Delivery Date. Click the Calendar to select the date and time.
  7. Click Find Distance/Time.

    Note: You can save this query as a Saved Query.

  8. To refine your query, click Refine Inquiry, and complete the above steps.
  9. To begin a new query, click New Inquiry, and complete the above steps.
  10. To view the log, click View Log.

The Distance/Time Results page lists:

  • Distance: The distance between two points in miles or kilometers, based on your user preferences.
  • Distance Type: The method that Oracle Transportation Management uses to lookup the distance: Lookup, Lookup Else Estimate, or Estimate.
  • Time: Duration for travel between two points, shown as hours, minutes, days, weeks, based on your user preferences.
  • Service Type: The Rate Service Type defined for this rate service.

    Note: The above columns only appear if you selected BOTH in the Request Type field.
