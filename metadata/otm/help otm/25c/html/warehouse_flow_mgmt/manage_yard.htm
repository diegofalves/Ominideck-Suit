Operational Planning

Manage Yard

This page is accessed via Operational Planning > Appointment Management > Dock and Yard Managers. Then select a location and run the Manage Yard action.

The Manage Yard action opens a page that displays all the shipments and shipment groups scheduled into slots in the yard. From a yard you can see the related appointments by clicking the Manage Appointments link in the upper-right corner. You can change the labels of the slots and rows in a yard either through changing yard properties or by setting your own labels on the location. The following applies to yards:

  • Shipments on other location resource types can be moved to a yard.
  • Shipments without appointments can be moved to the yard.
  • Shipments in the yard can be scheduled to other location resources as well as move to a different slot in the yard.
  • Once a shipment is moved to a slot in a yard that slot is unavailable for other shipments.
  • When a shipment is moved to a location resource other than the yard itself, the appointment on the yard stays until the appointment on the other location resources starts. That is, the yard will be unavailable to other shipments until the appointment on the other location resource starts.

The following properties exist to help when working with yards:

  • glog.otm.yard.defaultRowLabelPrefix: provides default prefixes for the rows in the yard.
  • glog.otm.yard.defaultSlotLabelPrefix: provides default prefixes for the columns in the yard.
  • glog.otm.yard.detentionTimeWarningRanges: defines the warning levels for excessive detention time for shipments in a yard.
  • glog.otm.yard.displayText.format: defines the reference number qualifier for the reference number.
  • glog.otm.yard.displayText.shipmentRefnumQual: configures the text that appears in the shipment fields on the Yard page.

Yard

Right-Clicking a Cell with a Shipment:

If you right-click a cell that is in the Yard table and has a shipment in it, you are given a list of available options.

  • Remove from Yard: When you select this action, the shipment is removed from the yard and appears in the unscheduled shipment list. If appointments have scheduled other types of resources for the shipment, this action will not do anything unless the appointments are removed first.
  • Schedule Appointment: Schedules the appointment, similar to the buy shipment agent action and the shipment group agent action.
  • Schedule Appointment - Show Options
  • View Details: When you select this action, the detail information of the shipment in the yard is displayed. If the appointment is scheduled but has not yet started, the end time of the occupied yard will be the appointment start time on other resources. Otherwise no end time will be available.
  • View Shipment Information: Displays the details of the shipment that you right-clicked.
