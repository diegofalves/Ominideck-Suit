Configuring Manager Layouts

This page allows you to configure a manager layout.

Note: Help is based on the standard product. You may have a configured version of OTM so fields may be hidden, tabs may be hidden, etc. Help will not reflect those configurations.

To configure Oracle Transportation Management to use a configured manager layout:

  1. Create a View (V) or Edit (E) Manager Layout using the Copy Manager Layout action.
  2. Edit the new manager layout via the Manager Layout: Detail page.
  3. Attach the manager layout you created to the appropriate screen set. For instance, attach a shipment manager layout to the Shipment screen set. Attach a manager layout to a screen set by going to the General tab within the screen set manager and select from the options available for each type of page (edit, new, or view). New and edit pages use the Edit type of manager layout. The view page uses the View type of manager layout.

    Note: While the Rapid Order Base and Rapid Order Release managers can be configured, they do not use screen sets so you can only add them to menus as external URLs. See the topic Menu Objects for details.

    Occasionally after upgrading your system, customized screens may need to be manually regenerated. If you have made customizations such as addingÂ  fields, you may need to remove and re-add them again to regenerate the screen.

  4. Attach your screen set to a new or existing menu layout. After a group or link has been created, select the screen set option button and attach your screen set with the new manager layout.
  5. Add the new menu option to the appropriate user by using the User Access Manager. The User Access Type should be set as "User Menu" with the appropriate User ID.
  6. When editing user access, select the menu layout you created.

    Note: If you have multiple servers, you must deploy the configured files to all of the servers.

  7. After logging out and back into Oracle Transportation Management, the navigation menu will have a menu option that links to the manager layout you created.
