Data Management: Business Intelligence

This page is accessed via:

  • Transportation Intelligence > Data Management > View FTI Data Load Log and View FTI Data Load Rejects
  • Global Trade Intelligence > Data Management > View FTI Data Load Log and View FTI Data Load Rejects

Note: You must be logged in as DBA.ADMIN to access this page and these processes.

See the topic Data Flow to Transportation Intelligence or Data Flow for Global Trade Intelligence for an explanation of the data flow.

Both View FTI Data Load Log and View FTI Data Load Rejects menu items open the System Dashboard which you can use as follows:

Load Log Tab

This tab enables you to view the data that uploaded along with possible errors.

The filters are on the top of the dashboard and include:

  • Table Start Date: Enter a start and end date. When you select a table start date, the Load ID filter populates with a list of load IDs that fall between the selected dates.
  • Load ID: Unique identifier for an ETL run.
  • Table Name

To use the filters,

  1. Select a Table Start Date from the filters at the top of the page.
  2. Once the Load ID drop-down populates, you can further filter by load ID.
  3. Click anywhere in the filters pane to refresh the data on the any dashboard tab.
  4. Click Reset to either reset the prompts to the default values or clear the prompts of all values.

Load Rejects Tab

Enables you to view the data that did not get uploaded.
