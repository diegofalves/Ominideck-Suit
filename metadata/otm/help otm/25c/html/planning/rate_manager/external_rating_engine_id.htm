Contract and Rate Management

List of External Rating Engine ID

Here is the list of external rating engine IDs and their description:

Note: Support for on-premise SMC3 RateWare applications has been deprecated and is not supported.

Rating Engine

Description
FEDEX

FedEx provides rate and transit application programming interface (API) with estimates for account-specific FedEx rates available on the origin/destination pair provided, including estimated shipping costs with discounts, surcharges, and other charges affecting your shipping rates. You will also get transit time along with rates.

Note:Â The FedEx integration has been deprecated and Oracle is in the process of replacing it with an integration that is supported by the FedEx Compatible program.

GENERIC

This is an OTM interface provided to any third-party rating engine which enables you to write your own Java class using the existing Oracle Transportation Management interface. This will allow a client-specific rating engine to meet your needs. For more information, please refer to the External Programming Interface Guide.

KEWILL

Note: BluJay Solutions Parcel (formerly Kewill Flagship Parcel Rates) is not supported.

Â 

This is an application server that is designed for consolidated parcel-mode shipping serving both domestic and international requirements. It automates document generation and provides carrier compliance screening.

SMC3 WITH CARRIERCONNECT

This is a socket server application that helps shippers to determine transit time and rates for LTL carriers.

RATEWAREXL

This is a Web service that enables you to apply discounts, minimum charges, and deficit rating during the shipment rating process. It offers Less Than TruckLoad (LTL) and density pricing solutions. It uses multiple data and tariff tables to provide flexible rating methods.

SMC3 (Rateware)

This is a socket server application (custom) that provides ratings for LTL shipments and small package freight shipping. It handles shipment cost and pricing and includes tariff data modules which are standard modules for many LTL freight carriers.

RATEWAREXL WITH CARRIERCONNECT

This is a web service that provides rating and transit time for LTL shipments. It calculates total actual cost and total weight cost of a shipment. If you have the RATEWAREXL WITH CARRIERCONNECT license, you can also have access to the CarrierConnect services. View this web service definition.
