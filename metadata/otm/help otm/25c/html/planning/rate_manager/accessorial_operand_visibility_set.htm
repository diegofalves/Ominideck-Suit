Accessorial Operand Visibility Set

This page is accessed via Contract and Rate Management > Contract Management > Rate Offering or Rate Record > Accessorial Operand Visibility Set.

The Accessorial Operand Visibility Set is a configuration object within your application which acts as a filter, providing a defined collection of specificÂ Rate Basis Items (RBIs). It allows you to control the list of available RBIs you can view and select when creating or editing accessorial costs. This ensures that only the RBIs included in this set will be displayed as options when associating charges and conditions with accessorial costs.Â Additionally, this feature helps you manage which RBIs are relevant for accessorial costing, potentially simplifying the user experience and ensuring data accuracy.

Adding anÂ Accessorial Operand Visibility Set

  1. Enter anÂ Accessorial Operand Visibility Set ID.
  2. SelectÂ the domain in which the accessorial operand visibility set will be active from the Domain Name drop-down list.
  3. Enter the Description of the accessorial operand visibility set.
  4. Create Accessorial Operand Visibility Set by selecting the required RBIs from Available.

For the existing accessorial costs, the application displays the currently saved RBIs along with the new visibility set in the dropdown. Once the new operand visibility set RBI is saved, older RBIs (which are not part of the new accessorial operand visibility set) will no longer be visible.

OnceÂ the Accessorial Operand Visibility Set is created, its ID must be configured in the Parameter Set associated with the user's preference. The parameter to be configured in the Parameter Set is "ACCESSORIAL OPERAND VISIBILITY SET".

Note:Â When the optional feature "ENABLE ACC COST RESTRICTED RBI VISIBILITY" is enabled, the application will activate the accessorial operand visibility set which would restrict the available RBIs for the accessorial costs. You can create this set by selecting the specific RBIs that should be visible on the accessorial cost screen.Â
