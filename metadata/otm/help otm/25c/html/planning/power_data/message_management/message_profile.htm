Message Profile

This page is accessed via Business Process Automation > Power Data > Message Management > Message Profile.

A message profile provides a grouping of stylesheet profiles for inbound message processing. It represents the different formats a device can send to Oracle Transportation Management. A message profile is assigned to a contact. If the message type for an incoming message, the system attempts to apply the stylesheet profiles listed in the message profile. They are applied to the incoming message in the order of the Attempt Sequence of the message profile.

Adding a Message Profile

  1. Enter a Message Profile ID.
  2. Enter a Description.

Message Profile Details

  1. In the Attempt Sequence field, enter a number to sequence the attempts at applying a stylesheet profile to an inbound message, thereby determining the appropriate profile.
  2. Enter a Stylesheet Profile ID.
  3. Click Finished.
