Incoterm Profiles

This page is accessed via:

  • Order Management > Power Data > Order Management > Incoterm Profiles
  • Master Data > Power Data > General > Incoterm Profiles

An Incoterm profile restricts the itineraries that can be used to plan shipments for an order, based on the Incoterm attributes specified on the order. After you create an Incoterm Profile, you must assign it to an itinerary and Oracle Transportation Management will either use or exclude itineraries based on the Incoterm on the orders being planned. In addition to checking the Incoterm, Incoterm Location text may also be optionally included with an Incoterm when it is added to the profile. If it is specified, it also filters itineraries.

For example, you could create an Incoterm Profile that contains a list of specific Incoterms and/or Incoterm Location text (and mark them Compatible). If you assign that profile to an itinerary, Oracle Transportation Management will consider that itinerary as a valid choice for shipment planning on any order that has all of the Incoterm attributes specified in the profile. If an order does not have the same Incoterms attributes then the itinerary is not valid (this also applies if the order has no Incoterm attributes).

Incoterm profiles can also be used to determine if non-freight related charge (NFRC) rules apply to a job. If at least one order release on a job has an Incoterm that is compatible with the Incoterm profile defined, then the criteria will be satisfied and the NFRC applied.

An Incoterm profile also allows you to create one compliance rule for a group of Incoterms, based on which a formula will be assigned and used for landed cost calculation.

Adding an Incoterm Profile

  1. Enter a unique identifier in the Incoterm Profile ID field. You will use this ID when you refer to this profile on other pages.
  2. Enter a name for the profile in the Incoterm Profile Name field.
  3. Select the Compatible checkbox to indicate that the Incoterms and/or Incoterm text included in the profile must also be present on the order for the itinerary to be a valid choice for building a shipment (for the order). If you do not mark the Compatible checkbox, all Incoterms except the ones in the profile are valid for itinerary-order matching.
  4. Enter an Incoterm ID in the Incoterm ID field.
  5. Enter a location in the Incoterm Location field.
  6. Click Save.

    Note: You can add as many Incoterm IDs as you need. Remember to click Save after entering each Incoterm ID.

  7. Click Finished.
