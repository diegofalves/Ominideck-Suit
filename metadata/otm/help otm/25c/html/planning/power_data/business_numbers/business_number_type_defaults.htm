Business Number Type Defaults

The Business Number Generator (BNG) automatically generates and assigns sequential numbers to certain ID and other fields. This table describes the business number types that are generated by default. You can change and create business number types by using Business Number Type Management. The context for is defined by the business number type. You cannot change the default BN types to have a different context.
