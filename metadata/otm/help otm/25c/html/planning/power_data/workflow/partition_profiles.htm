Partition Profiles

This page is accessed via Business Process Automation > Power Data > Event Management > Partition Profiles.

A partition profile is a group of partition IDs that can be used during bulk planning to make partition assignments to a group of orders based on the saved query defined on the partition.

You can also use a partition profile prior to bulk planning to make partition assignments using the Assign Partition process in Process Management.

Defining a Partition Profile

  1. Enter a unique Partition Profile ID.
  2. Select a Domain from the drop-down list.

Partition Profile Detail

  1. Enter a Partition ID to include in the profile.
  2. In the Seq No field, enter a sequence number for the partition ID. The sequence number determines the order in which to apply the partition queries in case any orders satisfy multiple queries on the partitions in the profile. For example:

  3. Click Save for each partition ID you enter.
  4. Click Finished.
