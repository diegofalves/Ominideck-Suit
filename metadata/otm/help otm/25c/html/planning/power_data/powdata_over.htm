Power Data

Power Data represents setup or configuration data that you should define early on in your Oracle Transportation Management implementation. In many cases, the definition of Power Data affects the way in which Oracle Transportation Management functions. For example, many of the Power Data options directly impact how you work with orders, shipments, service providers, locations, itineraries, rates, etc. The Power Data that you can pre-define in these areas can be used in many of the Oracle Transportation Management Managers (Order, Rate, Location, Material, Itinerary, Payment, etc.) to reflect your transportation management needs.

Note: Power Data is organized by component.
