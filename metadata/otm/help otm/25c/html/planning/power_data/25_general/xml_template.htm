XML Template

This page is accessed via Business Process Automation > Power Data > Integration > XML Template.

XML templates can be used when you are generating Out XML Profiles in place of specifying individual XML data to be included or excluded. For example, you could generate an XML template by generating it based on a location in your system.

  1. Enter an XML Template ID.
  2. Select an XML Element from the drop-down list.
  3. Select a Domain Name from the drop-down list.
  4. Enter a Description for the XML template.
  5. Select the Use Data checkbox to indicate that you want to constrain the data generated using the XML template by the criteria designated at the time of the XML generation. For example, you have a shipment with three different reference numbers on it and you want the XML to only contain a certain reference number, then you can indicate which reference numbers to place on the XML. If you do not have the Use Data checkbox on the XML template selected, then the special criteria will be ignored and all data will be placed in the generated XML.
  6. Click the Upload button to upload an XML template file to the system. You can upload XML files, such as .xml. The <?xml> header line preferred but nor required.
  7. Click the Export button to export the XML template.
