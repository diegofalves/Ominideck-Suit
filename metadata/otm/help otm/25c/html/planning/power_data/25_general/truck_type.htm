Truck Type

This page is accessed via Shipment Management > Power Data > General > Truck Type.

Truck types are a categorization of the equipment group. Several truck types are shipped with OTM including for example, Light Delivery Truck, Light Delivery Van, Straight Box Truck - Light, Straight Box Truck - Heavy, Straight Flatbed Truck, Dromedary, Pony Trailer and Semi-Trailer. You can add additional truck types, but not trailer types.

Use this page to create truck types. Truck types are used to determine the vehicle type for external distance engines, the length based on Pulling Length (which requires a default power unit in some cases, such as semi-trailer) and the height based on the roof type. The Pulling Length and Truck Type fields are used on equipment groups.

Out of gauge dimensions can be calculated based on roof type, trailer type and the Allow Side/Rear Overhang checkboxes. When selecting a truck type that does not allow overhang, or an enclosed truck type, no out of gauge calculations will be performed for the shipment.

Truck type details are included in calculation of loaded equipment dimensions for external distance engines. and is part of the configuration for sending equipment dimensions to external distance engines.
