Logic Configuration - Workbench

Logic configuration workbench parameters are accessed via the Logic Configuration page at Shipment Management > Power Data > General > Logic Configuration if you select a logic configuration type of Workbench.

Logic configurations of this type are used to specify parameters to control the behavior of the Workbench.

MAP VENDOR

General

6.4

Map vendor name.

Default is empty or blank.

When this parameter is set to the default of empty/blank, the properties for the map vendor specified during configuration are used to determine the default map vendor. To configure map vendors, see the following:

  • For HERE maps: here.app_id and here.app_code
  • For ALK maps: alk.api_key
  • For Oracle Maps Cloud Service (Oracle eLocation) maps: elocation.mapviewer url and elocation.elocation_url
  • MAPLIBRE maps: elocation.elocation_url

If none of the above are set, then the map will be blank.

USE DEFAULT EDE PARAMETERS

External Distance Engine

6.4.3

Determines whether to use the default external distance engine parameters for routing on the map. Values are:

  • TRUE: Use the default map provider's public external distance engine parameters for routing. For example for HERE Maps use the external distance engine parameters set in the HERE DEFAULT EDE ID logic configuration parameter.
  • FALSE: Use the shipment's external distance engine parameters for routing on the map.

Default value is FALSE.

USE TRAFFIC

General

6.4.3

Determines whether to use historic traffic for street routing on the map.

  • FALSE: Do not use traffic for street routing
  • TRUE: Use traffic for street routing

Default value is FALSE.

If both USE TRAFFIC and USER TRAFFIC PER STOP are set to TRUE, USE TRAFFIC PER STOP is used.

Note: For use with HERE and ALK maps only.

USE TRAFFIC PER STOP

General

19B

Determines what traffic related data to send to the map vendor:

  • FALSE: Send shipment level planned or estimated departure time.
  • TRUE: For each shipment stop send the planned or estimated departure time.

Default value is FALSE.

If both USE TRAFFIC and USER TRAFFIC PER STOP are set to TRUE, USE TRAFFIC PER STOP is used.

Note: For use with HERE and ALK maps only.
