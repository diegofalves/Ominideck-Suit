Logic Configuration - Workbench Gantt

Logic configuration workbench Gantt parameters are accessed via the Logic Configuration page at Shipment Management > Power Data > General > Logic Configuration if you select a logic configuration type of Workbench Gantt.

Logic configurations of this type are used to specify parameters to control the behavior of the Enhanced Workbench.
