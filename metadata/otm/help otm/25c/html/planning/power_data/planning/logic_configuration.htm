Logic Configurations

This page is accessed via:

  • Shipment Management > Power Data > General > Logic Configuration
  • Master Data > Power Data > Configurations > Logic Configurations

Logic configuration is a collection of related parameters and algorithms used by OTM. A configuration can handle different scenarios and be shared by different parameter sets.

Configurations with a logic configuration type of Multistop are associated with the Multistop Config ID parameter on the Parameter Set page.

Adding Logic Configurations

  1. Enter a unique Logic Configuration ID.
  2. Select a Logic Configuration Type from the drop-down list. When you select the type, the page reloads with relevant parameters. For an explanation of each logic configuration type, select one of the parameters below.

    Note: If you want to select another type, re-navigate to the page. Do not click Back.

  3. Optionally, enter a Description for the configuration.
  4. Select the domain in which this configuration will be active from the Domain Name drop-down list.
  5. Change the parameter values as desired. See the following topics for details on the logic configuration types and their parameters:

    • Continuous Moves

    • GTM EBS Integration

    • GTM Logic Configuration
    • GTM Product Classification UI Configuration
    • GTM Transaction Configuration
    • Machine Learning
    • Maps
    • Multistop
    • Network Routing
    • Purge Configuration
    • Rate Inquiry UI Configuration
