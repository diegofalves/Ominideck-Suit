Logic Configuration - UI Configuration

Logic configuration UI Configuration is accessed via the Logic Configuration page at Shipment Management > Power Data > General > Logic Configuration if you select a logic configuration type of UI Configuration.

Logic configurations of this type are used to specify parameters to control UI functionality. The glog.webserver.ui_configuration_default_logic_config property sets a default, system-wide logic configuration. This logic configuration can be associated via User Preferences, via the User Interface Configuration Preference user preference, for applicability at the user level.

AUTO SELECT WHEN ONE RECORD

Result Selections

21A

When your search returns only one result, it will be automatically selected. This enables you to immediately run actions against it. Similarly, when you save a record, the save results page will automatically select that record if only one record was saved.

EXPORT FORMAT

Finder

24A

This parameter allows you to control the type of Microsoft Excel output when exporting from the finder and finder results pages. The options are:

  • Excel: Exports results to an Excel workbook
  • Unformatted Excel: Exports results to a tab delimited Excel file.

Default: Excel

MAINTAIN SELECTIONS WHEN RERUN QUERY

Result Selections

21A

If you select records on a Search Results page, and then click the Rerun Query icon, the results will be automatically selected if they still match the parameters of the query.

ONE OF SEARCH INCLUDE SPACE

Finder

23C

Controls the behavior of the "One Of" operator on Finders. If set to the default of FALSE, you must enter multiple values as a comma-separated list. If set to TRUE, you can either enter multiple values as a comma-separated list or a space-separated list.

OPERATORS FOR NAVIGATION LINK SEARCH

Home Page

21A

When using the Global Search to find Navigator menu links, you can choose whether your search Begins With the text you enter, or Contains it.

Other search results in Global Search, such as for saved query results, use Begin With, regardless of this parameter value.

RETAIN NEW LINE FORMAT IN GRID TEXT BOXES

Manager Layout

21C

Controls if carriage returns/ new lines are retained in grid text boxes.

REMOVE_LEADING_TRAILING_SPACE_ON_PASTE

Finder

23C

If set to true, when you paste text into a field on a finder any leading blank spaces and trailing blank spaces are removed.

SHOW DOMAIN NAME FOR DUPLICATE VALUES

Drop-Down Lists

22A

Duplicate values in a drop-down list will have the Domain Name appended when set to TRUE.
