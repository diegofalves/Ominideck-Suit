Logic Configuration - Rate Inquiry UI Configuration

Logic configuration rate inquiry interface parameters are accessed via the Logic Configuration page at Shipment Management > Power Data > General > Logic Configuration if you select a logic configuration type of Rate Inquiry UI Configuration.

The options you select determine how the Ask Oracle Rate Inquiry page will display.

Note: For these parameters to apply, the parameter set must be in the PUBLIC domain. If you copy the parameter set to another domain, Oracle Transportation Management will not use the selected parameters.

Note: The Logic Configuration ID of your configuration must be linked to your user preferences for RIQ UI Configuration Preference.

User-defined RIQ UI Configuration

  1. Enter an ID for the user-defined configuration in the Logic Configuration ID field.
  2. In the Logic Configuration Type field, select Rate Inquiry UI Configuration. The window refreshes with additional details.
  3. If desired, enter a description.
  4. Select the domain in which this configuration will be active from the Domain Name drop-down list.
  5. Select and define values for any or all of the parameters listed in the table below.
  6. Click Finished.

Parameter

Default

Description and Parameter Value

RIQ UI CREATE SECONDARY CHARGES

False

If true, secondary charges will be included in the total actual cost and total weighted cost on the Rate Inquiry results page. Additionally, there is a column "Type", that is only visible when this parameter is true. It displays 'SC' when the option is secondary charge. Otherwise, it will be blank.

Also, in Rate Inquiry results, the secondary charges will display immediately after their related shipments.

Parameters: True or False

RIQ UI Per Result UOM Type

Null

Parameters:

  • Volume
  • Weight
  • Empty - if this is empty, per result will not be computed.

RIQ UI Per Result Volume UOM

Null

UOMs for volume such as liter. You can configure UOMs that are not standard. These can be added by CSV.

RIQ UI Per Result Weight UOM

Null

UOMs for weight such as kilo. You can configure UOMs that are not standard. These can be added by CSV.

RIQ UI Sort By

Cost

Enables you to sort costs by weighted and actual costs as well as cost per UOM and weighted cost per UOM.

Parameters:

  • Cost
  • Cost Per UOM
  • Weighted Cost
  • Weighted Cost Per UOM
