Logic Configuration - Machine Learning

Logic configuration machine learning parameters are accessed via the Logic Configuration page at Shipment Management > Power Data > General > Logic Configuration if you select a logic configuration type of Machine Learning.

Logic configurations of this type are used to specify parameters to control the behavior of Logistics Machine Learning.
