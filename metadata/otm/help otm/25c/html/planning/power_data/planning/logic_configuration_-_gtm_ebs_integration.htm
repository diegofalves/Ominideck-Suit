Logic Configuration - GTM EBS Integration

You can access logic configuration parameters for GTM-EBS integration via the Logic Configuration page if you select a logic configuration type of GTM EBS Integration.

By default, the stylesheets required to transform EBS XMLs to GTM XMLs are looked up in the file system. The parameters listed here will enable you to toggle the stylesheets between the file system and the stylesheet content table. These parameters will be included in the GTM EBS Integration parameter set to configure the GTM-EBS integration functionality.

Parameters

Parameter

New in Version

Description and Parameter Value

  • ITEM SYNC STYLESHEET

  • ORDER COMPLIANCE STYLESHEET

  • PARTY SYNC STYLESHEET

  • SHIPMENT COMPLIANCE STYLESHEET

6.4.3

These parameters will allow stylesheets to be looked up from the file system.

The default values of these parameters are, respectively:

  • DEFAULT_ITEM_SYNC_STYLESHEET

  • DEFAULT_ORDER_COMPLIANCE_STYLESHEET

  • DEFAULT_PARTY_SYNC_STYLESHEET

  • DEFAULT_SHIPMENT_COMPLIANCE_STYLESHEET

USE STYLESHEET CONTENT FROM TABLE

6.4.3

This parameter will allow the system to use the stylesheets from the stylesheet_content table.

Default: TRUE
