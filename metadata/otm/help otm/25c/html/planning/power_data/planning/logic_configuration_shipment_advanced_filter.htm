Shipment Management

Logic Configuration - Shipment Advanced Filter

Shipment advanced filter parameters are accessed via the Logic Configuration page if you select a logic configuration type of Shipment Advanced Filter.

When running the action to assign shipments to a driver, advanced filters further limits the set of drivers considered from the saved query. When assigning a shipment, there is an override advanced filter button to do overrides.

Parameters

Note that some of these parameters do not have a default assigned.

Parameter

Group

New in Version

Functional Description

Shipment Match Driver Retained Equipment

Shipment

6.1

If this parameter is TRUE, the equipment type the shipment needs must be the same as the equipment type the driver retained, or the shipment does not need equipment and the driver has no equipment.

If this parameter is FALSE, this criteria is ignored.

Shipment Maximum Distance from Driver

Shipment

6.1

Indicates the maximum distance allowed from the driver's next available location to shipment source location

Shipment Maximum Duration from Driver

Shipment

6.1

This is the maximum duration from the driver's next available time to shipment start time.

Shipment Minimum Distance from Driver

Shipment

6.1

The minimum distance from the driver's next available location to shipment source location

Shipment Minimum Duration from Driver

Shipment

6.1

The minimum duration from the driver's next available time to shipment start time.
