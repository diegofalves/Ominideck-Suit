Splittable on Order Release

The splittable checkbox is on the order release manager accessed via Order Management > Order Release > Order Release.

The option is also found at Order Management > Order Base. Then click the Line Item tab and then click New Line Item.

While creating an order release from an order base that has lines, the Splittable checkbox is set based on the selected Releasing Logic and Splittable checkbox on the order configuration as well as the Splittable checkbox on the order base line.

The following table shows settings and expected results for various scenarios.
