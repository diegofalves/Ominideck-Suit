Order Management

Release Lines

This action is accessed via Order Management > Purchase Order > Order Base > Actions.

Use this action to create a release instruction (and subsequently an order release) for any packaged item on an order. The Order Release can then be used to plan a shipment.

When you perform this action, a list of packaged items defined on the order appears with the following attributes:

  • Packaged Item ID is the Packaged Item ID you supplied when the item was created. If you did not supply a Packaged Item ID when you created the item, use the Item ID.
  • Source/Destination Location
  • Total/Released columns show the respective values for the total ordered and release to date values.
  • Priority: The priority defined on the order base header is copied to the order release header.

Note: This action does not work when your order configuration has the Aggregate Lines check box selected.

View Existing Release Instructions

Click Release at the end of any Packaged Item row to display the existing release instructions for the item, if applicable. The following attributes appear:

  • Release Number is a sequential number that is automatically assigned to a release instruction.
  • Release Instruction identifies whether the instruction was released by Item Count, Weight, or Volume.
  • Amount is the amount of the Release Instruction Type that you used (count, weight, or volume).
  • Source/Destination Location are the original source/destination locations on the order (you can override these with each new release instruction).
  • Release Date is the date the release occurred. You can specify a date on a Release Instruction to control when a ship unit is released.
  • Pickup/Delivery Dates are those specified on the ship unit.
  • Tag fields can be used to enter user-defined data values. These Tag values are copied to the line item on the order release that is created from the release instruction. These Tag values also replace any Tag values that you specified for the line item.
  • Processed icon identifies whether an order release was successfully created from a release instruction (icon with a check mark means successfully processed). Release instructions are processed when you complete the Release Ship Unit action. When you create a new Release Instruction described below, this page re-appears after it is recorded. The Processed icon is set to unprocessed and you can edit or delete that Release Instruction. However, once the Release Instruction is completely saved, you cannot edit or delete it.

Create Line Item Release Instruction

  1. Click New Release Instruction and complete the Release Instruction page.

    Repeat to add as many release instructions as you need. You can also edit or delete a Release Instruction at this time since they are not processed. However, you cannot modify or delete a release instruction once it is saved.
  2. Click OK to display the Packaged Item list page again.

    You can record additional Release Instructions for any line item listed. Release instructions adhere to the same bundling rules as ship units. Therefore, the sequence in which you record them determines how order releases are created. For example, if you create one release instruction for one line item, Oracle Transportation Management creates one order release. However, if you create release instructions for several line items, Oracle Transportation Management tries to bundle the compatible line items on one release based on bundling rules. When packaged items are incompatible (for example, different Source/Destination Locations), Oracle Transportation Management creates separate order releases as needed for each.

    Click Finished to create the appropriate order releases based on the release instructions that you entered (unless you specified a Release Date for the release instruction that has not occurred).

The Order Releases that are created appear with the following attributes:

  • Order Release ID identifies the new order release that was created.
  • Total Weight/Volume for all line items on the order release.
  • Source/Destination Location and City
Click the Order Release ID to view the details, edit the order release, or perform Order Release actions.

Other Releasing Actions

The following Release Actions are also available for releasing line items. You can run these actions from the Order Base Line Item Manager.

  • Release Remaining Order Amount
  • Release Total Order Amount

Line Item Release Status

As you create release instructions for a line item, Status values for the Status Type LINE RELEASE STATUS are assigned.
