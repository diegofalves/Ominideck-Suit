Order Management

Ready to Ship

This page is accessed via:

  • Order Management > Purchase Order > Order Base Line > Actions
  • Order Management > Purchase Order > Order Base Ship Unit > Actions

You can use the Ready to Ship action to generate order releases from release instructions added on the Ready to Ship, Order Base Line, and Order Base Ship Unit pages.

The Ready to Ship page allows you to perform ready to ship to generate order releases and to make adjustments to any or all of the following: ship from location, ready to ship date, ship unit counts, weights, and volumes. Any adjustments you make apply only to the first, unprocessed release instruction. If all release instructions appearing on the Ready to Ship page are processed, then a new release instruction is created for the values you entered.

Use this page to create order releases based on the input of specified quantities against packaged item quantities. This page displays specific order information that vendors would typically want to review before releasing items for shipment. Various fields allow you to override the quantity of a line item/ship unit to ship, where to make the pickup, and to provide additional information in the form of remarks.

Note: Information from the order base is used when creating order releases from this screen. For example, the Release Method field on the order base is used when releasing items from this page.

This action can be run for both order base lines and order base ship units. There are a few differences between the Ready to Ship page that appear for an order base line and the one that appears for the order base ship unit. The first section below pertains to the order base lines.

Note: This action does not work when your order configuration has the Aggregate Lines check box selected.

Order Base Lines

The page also has an order base line section so that you can view the information and enter values that will be used to create the release instruction.

  1. To change the transport handling unit for the order release, select a Transport Handling Unit from the drop-down list.
  2. Enter a Ship Unit Count for the order release.

    Note: The transport handling unit and ship unit count fields only appear if you have selected a release method that has the Single Transport Handling Unit releasing logic on the order base.

Order Base Line Details

Here you can see the order base line, the packaged item on that line, and the item on that line. Also, you can see the total package count and the number of packages that were released. The Remaining Quantity column shows calculated values based on previous release instructions that were created for the order.

The following fields are populated if the release instruction from the order base line has the corresponding value.

  1. Enter an Early and Late Pickup DateÂ if you want to specify different pickup dates for the release instruction that is used to generate the order release.
  2. Enter an Early and Late Delivery Date if you want to specify different delivery dates for the release instruction that is used to generate the order release.
  3. If you want to specify a different ship from location than is indicated on the release instruction used to generate the order release, select a location to Ship From.
  4. If you want to specify a different ship to location than is indicated on the release instruction used to generate the order release, select a location to Ship To.
  5. To specify the number of items released by the release instruction, enter the Total Package Count for the release.
  6. Enter the Total Weight and Total Volume for the release. If you skip these fields, Oracle Transportation Management can calculate the values depending on the attributes of the original order.

Release Instruction History

The release instruction section contains processed release instructions.

Ship Unit

You can also run the Ready to Ship action for order base ship units. The Ready to Ship page that appears for order base ship units is slightly different than the one for order base lines. Unless otherwise noted, the function behind the fields is the same as described in the previous section. The Total Weight section displays the total weight of all ship units involved. The Total Volume field displays the total volume of all ship units involved.

  1. Enter an Early and Late Pickup Date.
  2. Enter an Early and Late Delivery Date.
  3. Select a location to Ship From.
  4. Select a location to Ship To.
  5. Select a Transport Handling Unit from the drop-down list.
  6. Specify the Gross Weight and Gross Volume Per Ship Unit.
  7. Specify the Length, Width, and Height Per Ship Unit.

    Note: The Total Package Count, Total Weight, Total Volume, and Calculate Total fields are not displayed when there is more than one ship unit content per ship unit.

  8. The Copy Values button copies the length, width, and height to the second order base ship unit that was selected when you ran the Ready to Ship action. By clicking the Copy Values button in each ship unit section, you can effectively copy the first section's values to each of the ship units, if you have selected more than two ship units.
  9. The Calculate Totals button populates the Total Weight and Total Volume fields for this ship unit. It also populates the Total Weight and Volume fields.
  10. Enter the number of ship units you want to release in the Ship Unit Count field.
  11. Enter the total package count.
  12. If you do not want to automatically calculate the Total Weight and Volume fields, manually enter the values here.

Release Instruction History

This section contains processed release instructions.

Order Release Remarks & Reference Numbers

You can also record additional remarks and reference numbers that are saved on the order release that is created when you complete this page.

Note: Remarks that your record are not saved on the original purchase order.
