Order Management

Workbench Move Order Movement to Shipment

This action enables you to add one or more existing order movements to a shipment on a workbench layout.

Note: This action is only available from the shipment table on a Workbench layout.

Drag-and-drop on a Workbench Layout

You can select one or more order movements from an order movement table and drag-and-drop them to a single shipment on a shipment table.

  • When one or more order movements are dropped on a shipment, the action Move Order Movement to Shipment runs in the background.
    • The affected shipments and order movements are updated on any tables, maps, or Gantt charts included in your layout.
    • If the action fails, you see the OTM error message.
  • The action runs with the following limitations:
    • The Consolidate Equipment option is false so Ignore Equipment Capacity is ignored and the equipment is added to the existing shipment. In this case, Oracle Transportation Management attempt to fit the goods of your order movement onto the existing equipment on the shipment.
    • All criteria are considered. You do not see the Planning Ignore or Fleet Ignore pages and cannot ignore any criteria.
