Order Movement Actions and SmartLinks

This page is accessed via Order ManagementÂ > Order Movement > Order Movement > Actions.

There are order movement actions that build shipments out of order movements. Set the parameter "IGNORE ORDER MOVEMENT OPERATIONAL DATES" to 'False' for these: e.g., actions such as Bulk Plan, Build Shipment on Leg, Build Shipment Show Options, Build Multi-Stop Shipment.

Note: If you are working with an Order Movement Template, the Operational Planning actions and the Order Management Change Order actions (with the exception of the Delete Order Movements action) will not run.

Note: If you are working with a network routable (parent) order movement, the Operational Planning actions (with the exception of Bulk Plan and Unassign Order Movement actions) and the Order Management Change Order actions will not run.

SmartLinks

From the Order Movement Manager results page, right-click on any ID to launch the following SmartLinks.

SmartLink

Displays

Documents

Documents results page showing Document IDs that are linked to this order movement.

Full Audit

Audit Trail displaying all changes that were made to the order movement.

Related Buy Shipment

Returns the buy shipments of the selected order movement. If the selected order movement is a parent order movement and is planned, this returns the shipments of the child order movements.

Related Network Leg Order Movements

Displays network leg order movements to which this order movement is linked.

Related Order Release

Displays the order releases to which this order movement is linked.

View Related Jobs

Job results page showing the jobs to which this order movement is linked.

View Related Shipment Groups

Displays the shipment groups associated with this order movement.
