Order Management

Calculate Best Direct Cost

This page is accessed via:

  • Order Management > Order Release > Order Release > Actions > Order Management > Calculate Best Direct Cost
  • Brokerage and Forwarding > Booking > Freight Forwarding > Actions > Calculate Best Direct Cost

Use this action to calculate the best (cheapest) direct cost for an order without creating a shipment. You must have the appropriate rates, itineraries and other shipment planning data in place for this action to work since Oracle Transportation Management initiates the standard shipment planning algorithms to calculate the cost. However, Oracle Transportation Management does not build a shipment for the order.

A results page displays the Order Release ID with the calculated cost. The Rate Offering that was used also appears. Oracle Transportation Management also populates this data in the appropriate fields on the Order Release Edit and View pages.

You can perform this action even if the order release is already planned on a shipment. This allows you to compare the cost of shipping direct verses other shipping options (bulk planning or multi-stop) that may have been used for the order.

Note: You can also use this action in agents.
