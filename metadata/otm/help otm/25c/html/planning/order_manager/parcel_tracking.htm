Order Management

Parcel Tracking

OTM enables you to identify and track individual parcels - whether or not they are part of a larger shipment. This is accomplished through the use of reference number qualifiers.

Typically, parcel tracking takes place at one of the following levels:

  • Order Base
  • Order Base Line
  • Order Base Ship Unit
  • Order Release
  • Order Release Line
  • Order Release Ship Unit
  • Shipment

Reference Number Qualifiers

Reference number qualifiers are defined in Power Data. Presently, there are coded reference number qualifiers named Tracking_Number for order bases, order releases, and shipments. However, you can define additional qualifiers.

For example, if you wish to add tracking numbers for an order base ship unit, first navigate to Order Management > Power Data > Qualifiers > Order Base Ship Unit Reference Number Qualifier.

  1. Perform a search to see if a tracking number qualifier already exists. If it does not, click New to create one. Be sure to select the Allow Visibility on Related Shipments option.
  2. Click Finished.
  3. Now open the Order Base Ship Unit manager. In the Reference Numbers section, select the Reference Number Qualifier ID of Tracking_Event (the one you created in step 1).
  4. Enter the tracking number in the Reference Number field.
  5. Click Save. You can enter as many reference numbers as you need.
  6. Click Finished.

Tracking a Parcel

You can use an automation agent to track a parcel if the parcel has a reference number qualifier attached to it. When you build the agent, select Tracking Event as the agent type. Then choose the tracking agent action Match Object by Tracking Number.
