SmartLinks

Right-click on any ID to launch the following SmartLinks.

SmartLink

Displays

Booked/Shipped/Received

Booked/Shipped/Received page that typically appears when running the Booked/Shipped/Received action against an order in Visibility.

Documents

Documents results page showing Document IDs that are linked to this order

Full Audit

Audit Trails results page displaying all changes that were made to the order.

Order Life Time Visibility

Navigation page so you can review the various stages of the order. Also, you can view related orders.

Original Order Base

Order Base Results page listing the order base from which the order release was created.

Related Bills

Bill Results page listing the bill records that were generated from the order release.

Related Buy Shipments

Buy Shipment Results page listing the shipments that were created from the order

Related Itinerary

Itinerary Results page listing the itinerary that was assigned to the order release as a result of shipment planning.

Related Rate Record

Rate Record Results page list the rate record that was used to build a shipment for this order release. If the order release is not planned on a shipment, no rate records are displayed.

Related Sell Shipments

Sell Shipment Results page listing the shipments that were created from the order.

Track and Trace

Track and Trace page that appears when you perform the Track and Trace action against an order in Visibility.

View Appointment

View page showing various ship unit data in addition to the pickup/deliver appointment dates and appointment number.

View Financials

Financial page so you can review the current bills and transportation costs that resulted from an order base/order release.

View Related Commercial Invoices

Commercial invoices created for the order release.

View Related Jobs

Jobs to which this order is assigned.

View Related Order Movements

Order movements created from the order release.

View Related Quotes

Quotes that were used to create the order release.

View Related Reference Numbers

Hierarchy of reference numbers from related business objects.

View Related Remarks

Hierarchy of remark records from related business objects.

View Release Instruction Totals

Ordered and released totals of the line item or ship units release instructions that were used to create the order release.
