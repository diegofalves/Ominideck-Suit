Freight Forwarding: Other Attributes

This page is accessed via Brokerage and Forwarding > Booking > Freight Forwarding. Click the Other Attributes tab.

Many of the attributes that appear on this page are similar to those available on the order base and order release managers.

This help page only describes the attributes that are specific to the freight forwarder order release. Click the following links for information on the other attributes that are also applicable to an order base and standard order release.

  • Buy Side/Sell Side General Ledger Code
  • Stuffing/Destuffing Location
  • Voyage Stowage Mode
  • Unitization Condition Name/Customer Unitization Request
  • Best Direct Rate/Cost

Freight Forwarder - Specific Attributes

Ultimate Country of Destination is the final destination country for an order. For instance, you could be responsible for an order between France and Egypt, but the final destination of the order is in the United States. This field is for information only.
