Brokerage and Forwarding

Quote Find Shipping Options

The Shipping Options page displays a list of potential shipping options. Use the checkboxes next to each option to select those that you want to generate quotes for. This page is accessed via the Quote Request tab on the Quote Manager.

After you have made your selection, click Add To Quote to proceed to the Cost Options tab.

Shipping Information

The options are listed from the least expensive to the most expensive. The list may include options that do not meet the requested service time. You may want to consider them if the cost is cheaper and your customer's service time requirements are flexible.

  • Options that do meet the service time requirements are identified with the hour-glass icon .
  • Options that do not meet the service time requirements are identified with the red-x hour-glass icon .

Each valid shipping option is identified by the Expand icon. Clicking this icon shows more detailed information.

The detailed information for each option includes everything outlined in the table below. On the page, some of the data are links. If you click on them, detailed information about that particular piece of data opens. Examples of such links are Service Provider, Origin, and Destination.

Itinerary

  • Total Actual Cost: The total cost of the shipment.
  • Transit Time: The total transit time. If there is only one leg, the two transit times are the same.
  • Transport Mode: The transport mode of the master leg.
  • Service Provider: The service provider of the master leg.
  • Start Time: This is the scheduled start time.
  • End Time: This is the scheduled end time.

Details

  • Source Location ID: The origin location.
  • Source Arbitrary: The source arbitrary location. These represent interim shipping locations identified in the Itinerary Manager and the Rate Manager. The Source Arbitrary locations are generated as shipment stops only when an arbitrary rate is used.
  • Port of Load: The port where the shipment is loaded.
  • Port of Discharge: The port where the shipment is unloaded.
  • Destination Arbitrary: The destination arbitrary location. These represent interim shipping locations identified in the Itinerary Manager and the Rate Manager. The Destination Arbitrary locations are generated as shipment stops only when an Arbitrary Rate is used.
  • Destination Location ID: The destination location.
  • Total Actual Cost: The total cost for this particular service provider.
  • Transit Time: The transit time for this particular service provider.
  • Transport Mode ID: The transport mode for this particular service provider.
  • Service Provider ID: The service provider of this leg of the shipment.
  • Preferred Carrier: If selected, it indicates that this carrier is preferred.
  • Start Time: This is the scheduled start time.
  • End Time: This is the scheduled end time.
  • Commitment Allocation: Planned allocations are percentage commitments from your carriers.
  • Commitment Count: This is the amount that has been committed from your carriers.
  • Quote: This checkbox is selected if this is a quote.
  • Rate Record: The specific rate record used.
  • Rate Offering: The specific rate offering.
  • Equipment Group: The equipment group for the shipment.
  • Expires: The rate's expiration date.
