Brokerage and Forwarding

Quote Request

The Quote Request page allows you to search for valid rates in order to provide the customer with the quote they requested. You can enter as much or as little information as you like. The only fields that are required are the Origin Location and Destination Location fields. Note that the more information you enter for your search, the more accurate the results are. If you leave a field blank, all choices are considered for that data option. This page is accessed via the Quote Request tab on the Quote Manager.

The searchable options available to you in the Quote Request page are split into the sections below.

Entering Quote Request Details

  1. Enter an Origin Location ID and a Destination ID.

    OR

    If you have the glog.quote.locationType property set to True, enter the following:
    1. Enter an Origin Location Type and a Destination Location Type. This limits the location search to these specific types of locations. The default Location Type is Location Code for both the Origin and Destination.

      Note: If the glog.quote.locationType property is set to FALSE, then the Location Type options will not be shown.

    2. Enter both an Origin Location ID and a Destination Location ID. If you select City/Zip as your Origin Location Type or your Destination Location Type, the Location ID fields are replaced with City, Province Code, Country, and Zip fields. Enter the appropriate information in each of these fields.
    3. Alternatively, you can use the VIA Location fields when generating quotes for international shipments. This limits the results to those with matching VIA Origin or Destination Locations.

Shipping Information

You can refine your search by entering specific shipping information. There are two types of shipping information you can specify: Commodity and Ship Unit (transport handling unit).

Commodity

  1. Select a Commodity from the Flexible Commodity Qualifier drop-down list.
  2. Enter a commodity code.
  3. You can further refine this commodity search by specifying the Volume or Weight of the commodity you are shipping. Enter the volume or weight value, followed by the appropriate unit of measure.
  4. Select the Splittable checkbox if the shipment can be split to use multiple pieces of equipment.
  5. Select a Transport Handling Unit from the drop-down list. The values in the drop-down list match the Transport Handling Units defined in Power Data.
  6. To refine your Transport Handling Unit search, specify the number of Transport Handling Units you are shipping in the Transport Handling Unit Count field, as well as the Length, Width, Height, and Declared Value of the Transport Handling Unit. You can list as many Transport Handling Units as you want.
  7. To enter more, click Save and repeat the steps.

Hazardous Cargo

  1. If you are searching for international shipping rates for items that are defined as hazardous, select Yes.

Perspective

  1. Select the appropriate Perspective for the quote you are generating; Buy for buy shipments, Sell for sell shipments.

Rates

  1. Select Include Domain-specific Rates to include rates that contain a domain profile that is the same as the quote's domain.

    OR

    Clear the checkbox to find rates that do not contain a domain profile and are therefore valid for all customers.
  2. From the Transport Mode drop-down list, select the mode that you want the master leg of the shipment to be transported on. The values in this drop-down list match the Transport Modes defined in Power Data.
  3. In the Service Provider ID field, enter or search for the service provider that you want to use for the master leg of the shipment. The values available in this field match what is defined in the Service Provider Manager.
  4. If you want, enter a Consolidation Type ID. This is optional. If entered, it is used by the Non-Freight Related Charge (NFRC) rule look up logic.
  5. Enter a date in the Available By field if the shipment will not be available to ship until a certain date.
  6. Enter a date in the Deliver By field if the shipment must be delivered by a certain date.

Options

In this section, you can select Optimize, Equipment Groups, or Equipment Configuration.

  • Optimize returns quotes that match the search criteria. This is the default setting.
  • Equipment Groups limit the search to specific Equipment Groups. When you select this option, an Equipment Group field appears where you can enter or search for an Equipment Group. Use the Save button to enter multiple Equipment Groups.
  • Equipment Configuration works like the Equipment Groups option, but it provides you with a field to enter the number of Equipment Groups you require.

Find Shipping Options

  1. Click Find Shipping Options to produce a list of valid shipping options for which you can generate quotes.
