Order Management

Order Base Line Item

This page is accessed via:

  • Order Management > Purchase Order > Order Base. Click the Line Item tab. Click New Line Item.
  • Order Management > Purchase Order > Order Base Line.

Assign a packaged item to the order by completing the appropriate fields that appear on the page. Provide as much detail as possible since Oracle Transportation Management uses Â this data in the planning process to bundle orders and create shipments.

Adding Order Base Line Items

  1. Parent Order Base displays the ID of the order base to which this line item is assigned. This field only appears on the Order Base Line Manager page.
  2. Enter a Packaged Item and Item ID. One of these is required; however, if only the item is entered, an auto assign rule should be configured to set the packaged item.
  3. Enter the Total Package Count.
  4. Select a Packaging Unit.
  5. Enter the Packaging Unit Count.
  6. Enter the Ready Quantity value to specify the quantity of the packaged item on the line that is ready to be shipped.
  7. The Source and Destination Locations identify where the item originates and where it is shipped. Oracle Transportation Management uses these fields to identify appropriate itineraries and service providers during shipment planning.
  8. Plan From/To Location fields let you override the default alternate location to or from which an order will be sourced/delivered for transportation planning purposes.
  9. Load/Unload Points are specific loading and unloading points assigned to a location. You can set load/unload points for the Source/Destination locations as well as the Planned From/To locations on an order. They are copied to shipments built from the order and displayed on the Shipment Stop tab.
  10. Enter a Buffer Location if you want to stage your shipment in a Â location different from Â your plan from location.
  11. The Order Release Line ID is a link to the order release line that was created from this line.
  12. Splittable describes whether a line item can be split into multiple ship units and placed on multiple shipments or multiple equipment when the total weight or volume of the line item cannot be handled by the Equipment Group associated with the shipment during planning. You can configure the itinerary to determine whether Oracle Transportation Management creates multiple shipments or uses multiple equipment.
  13. Shippable controls whether the packaged item can be automatically released for the full amount. By default, this checkbox is clear - meaning that the packaged item is not released when the order is saved. This checkbox is not available when you have selected the Aggregate Lines checkbox on the order configuration.

    Note: Release instructions created via the Shippable flag are based on the value (count, weight, or volume) that is set for the Releasing Qual parameter (the default value is Weight).

  14. Enter the Weight and/or Volume. Choose a unit of measure for each.
  15. Enter the Total Declared Value.
  16. Early and Late Pickup and Delivery dates can be used to determine the time window in which the line item can be picked up from its place of origin and delivered to the consignee. Oracle Transportation Management also uses these dates to determine whether multiple line items can be bundled into one order release. The pickup and delivery dates for multiple line items must overlap if you want to bundle them on one order release.

    Note: When the Early Pickup field is populated and the Hold As Late As Possible constraint is true, Oracle Transportation Management will no longer allow shipments to plan in the past unless the Date Emphasis constraint is set to PAST.

  17. Select the Pickup Appointment option to copy the early pickup date to the late pickup date.
  18. Select the Delivery Appointment option to copy the early delivery date to the late delivery date.
  19. The Early and Late Pickup/Delivery dates can also be used to set appointments.
  20. Enter the Number of Layers on Transport Handling Unit.
  21. Enter the Quantity per Layer.
  22. Select a Transport Handling Unit to define what the item is shipped on, such as a pallet. Transport handling units are a type of ship unit specification.
  23. Select a Transport Handling Unit Profile.
  24. If you want to round the line item, choose a Rounding Instruction from the drop-down list.
  25. Choose what you want to round to by selecting an option from the Round To drop-down list.
  26. The Released (Weight, Volume, Count) are the applicable values that result from releasing packaged items (based on existing release instructions).
  27. Quantity Remaining is the difference between the Total Package Count and the Released Count.

    Note: This can be a negative number if the order is marked for over-releasing.

Release Instructions

This section of the page summarizes the existing release instructions that have been created for the packaged item (if there are any).

Click New Release Instructions.

Note: You can also use the Release Lines action to create a release instruction and order release.

Equipment Reference Units

You can select Equipment Reference Units (ERU) to limit the number of specific units being loaded on a specific Equipment Group. If any ERU level information is provided on the ship unit, it overrides any defined on the ship unit specification.

  1. Enter an Equipment Reference Unit ID.
  2. Enter the number of reference units in the Equipment Reference Units Per Ship Unit field.
  3. Enter a value in the Total Reference Units field.
  4. Click Save for each equipment reference unit you enter.

Reference Numbers

Use this section to assign reference numbers.

Involved Parties

Use this section of the page to assign contacts/contact groups as involved parties on the order.

Attributes

Enter an Item Attribute Qualifier and Value that describes the feature for the item. For example, you could define an Item Attribute Qualifiers such as color, size or style for items and enter the appropriate value that makes sense for the qualifier.

Note: Feature attributes are copied to the Order Release Line Item when orders are released. These attributes do not appear on a shipment that was planned from an order release.

Note: You can also assign Item Attribute Values and Qualifiers on the item in the Material Manager.

Remarks

  1. Use this section to record remarks and remark qualifiers.
  2. Enter Tags.
  3. Click Save. You can add multiple remarks.

Packaging Reference Units

  1. Enter an ID in the Packaging Reference Unit ID field.
  2. Enter a value in the Total Packaging Reference Unit Count field.
  3. Enter a value in the Unit Packaging Reference Unit Count field.
  4. Click Save. You can add multiple packaging reference units.
