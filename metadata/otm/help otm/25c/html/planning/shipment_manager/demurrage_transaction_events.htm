Demurrage Transaction - Events

This page is accessed via Shipment Management > Visibility > Demurrage Transaction. Select the Events tab.

This page captures details of the start and end event. Also captures all the events received along with their date and time. If the agent setup is done, it is automatically updated every time a matching demurrage event is received. It can also be added/edited manually.

Start Event Details

Enter the details for the start event.

  1. Enter a Railroad Reporting SCAC.
  2. Select the Car Loaded checkbox unless the car is empty.
  3. Enter an Event Status Code ID.
  4. Enter the Location ID.
  5. Enter a Time Zone.
  6. Enter the City, Province Code and Country.
  7. Enter the SPLC.

End Event Details

Enter the details for the end event.

  1. Enter a Railroad Reporting SCAC.
  2. Select the Car Loaded checkbox unless the car is empty.
  3. Enter an Event Status Code ID.
  4. Enter the Location ID.
  5. Enter a Time Zone.
  6. Enter the City, Province Code and Country.
  7. Enter the SPLC.

Activity Type

The Activity Type grid will only be populated after running the Calculate Charges action. The activity type is determined by the action, such as Load to Load, Load to Empty and so on.

Event Date & Time Information

The event time displays the local time zone of these events instead of the user-preference time zone when possible:

  • If the time zone in the Location section of the Demurrage Transaction Header is populated, it is used.
  • Or, if the Location ID on the header is populated and has a corresponding time zone, it is used.
  • If neither of the above are populated, it defaults to the user-preference time zone.

Enter the date and time details for the event.

  1. Enter the Event ID.
  2. Enter the Event Date/Time.
  3. Click Save after each event added.
