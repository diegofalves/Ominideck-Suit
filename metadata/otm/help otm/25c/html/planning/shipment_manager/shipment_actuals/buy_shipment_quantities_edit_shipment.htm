Buy/Sell Shipment Quantities - Shipment

This page is accessed via Shipment Management > Shipment Management > Buy or Sell Shipments > Actions > Shipment Management > Utilities > Edit Quantities. On the Buy/Sell Shipment Quantities screen, select Edit By Shipment and Edit By Ship Units or Edit By Line Items and then click Submit.

Identification

The top section of this window contains read-only information about the present shipment.

Shipment Equipment

The Shipment Equipment section displays information about equipment that is already assigned to the shipment. If you wish to modify this information, click the Edit icon beside any shipment equipment line. A new screen opens. Â Its content depends on your selections in the Buy/Sell Shipment Quantities screen. Those screens are as follows:

  • Buy/Sell Shipment Quantities - Shipment Line Item
  • Buy/Sell Shipment Quantities - Shipment Ship Unit

Add Equipment

To add more equipment to the order release, enter a value in the Equipment ID field and then click Save. If you wish to create a new Equipment ID, click the New icon beside the field to open a creation dialog.

Note: You will only be able to add new equipment in the Buy Shipment Quantities - Edit Shipment screen if you have selected None in the Propagation window (see step 2 in Buy/Sell Shipment Quantities) or if you selected the ALWAYS SHOW ADD EQUIPMENT OPTION parameter, or if you selected the Always Show Add Equipment Option checkbox. If you selected any other propagation value, the Add Equipment section will not appear.

When you are finished viewing/editing, click Finished.
