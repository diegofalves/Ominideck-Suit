Update Open Tender

This action is accessed via Shipment Management > Shipment Management > Buy or Sell Shipments > Actions > Shipment Management > Open Tender > Update Open Tender.

Using this action, you can updateÂ the response time and/or add service providers to an existing open tender. This action is applicable only:

  • When the shipment has an open tender recordÂ (Spot Bid/Broadcast)
  • When the tender status is SECURE RESOURCES_TENDER RESPONSE OPEN

Expected Response Time

You can update the response time for all the tendered service providers. The tendered service providers will be notified about the change in response time. Notifications are not sent to the service providers who decline the initial tender offer.

Note:Â The Expected Response Updated checkbox marked on the results page indicates that the response time is updated. TheÂ Expected Response UpdatedÂ field is available for customization for Online Booking and Tendering/Spot Bid screen sets.

Note:Â If you run this action, OTM updates the tender record as per the existing shipment data only. To modify the shipment data, use the action Modify Open Tender.

Add Service Provider

You can add service providers to the initial tender (refer toÂ Broadcast Tender and Spot Bid Tender to add service providers). The new service providers will receive the spot bid/broadcast tender notification. The response time for the added service providers will be the same as that of the initial tender (in case you do not update the response time).

Note: If the user preference "Disable Service Provider Selection For Open Tender" is set to yes, OTM does not allow you to add any service provider.Â 

Lane Service Providers

This section displays the list of lane service providers that are not tendered. You can select lane service providers that you want to add to the initial tender (refer toÂ Broadcast Tender and Spot Bid TenderÂ for more details on Lane Service Providers).

Note:Â You can update theÂ response time and add service providers simultaneously. The updated response time will be notified to the existing service providers, and the new service providers will receive the spot bid/broadcast tender notification.

Note: If the user preference Disable Service Provider Selection For Open Tender is set to yes, selection of service providers will be disabled.

Tendered Service Providers

This section displays the list of all tendered service providers.

Note: While running Open Tender actions, if a service provider had previously declined an ordinary tender of the same shipment, or if the tender had timed out, exclude those service providers from the list of service providers for Open Tender by setting the property glog.webserver.tender.spotBid.excludeCarriersRejectedTender to true.

If required, you can add the excluded service providers using Service Provider ID only (in the Add Service Provider section), and not with Saved Condition or Saved Query.
