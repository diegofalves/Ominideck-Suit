Shipment Management

Revert Swap

This page is accessed via Shipment Manager > Shipment Manager > Buy Shipments > Actions > Fleet Management > Manage Relays > Revert Swap.

Note: This action applies to swaps only. If you want to revert/reverse relays or breakdowns, use the Append Shipment action.

Note: This action does not work static trip, dynamic trip, flight, voyage, ground service, or depot shipments.

Note: This action does not work on equipment repositioning shipments.

Relay actions split two shipments into four shipments. Use this action to revert these four shipments back to the original two.

Note: This action will function for CLOSED-DETACHABLE shipments, but not for STATIC or DYNAMIC shipments.

Finding Shipments to Revert

Following is the criteria to determine shipments to revert:

  1. Stop requirements:
    • SH1 - last stop is a drop-off
    • SH1 last stop and SH2 first stop must be the same location
    • There can be no MOVE_EMPTY or CHECK _INVENTORY special services at the last stop for SH1 or the first stop for SH2, since these special services are not supported at intermediate stops.
  2. Status requirements:
    • SH1 must not be tendered
    • SH2 must not be in-transit
  3. Equipment requirement 1:
    • SH1 and SH2 should have only one freight s_equipment (IS FREIGHT=true)
  4. Equipment requirement 2 (this is only required for relay, swap or multi-leg shipments that share the same s-equipment):
    • SH1 and SH2 has the same freight s_equipment (Is FREIGHT=true)

For shipments to be reverted, the logic looks to find shipments that satisfy all 4 constraints; either a single shipment (for relay, swap or multi-leg shipments that share the same s_equipment) or nothing. If nothing is found, it will try to find shipments that meet the first 3 criteria (for breakdown or general shipments).

Revert a Swap

  1. Select one of the original shipments in the Shipment Results page.
  2. Click the action Revert Swap. The Revert Swap page opens. OTM finds and displays the swapped shipments. The page also displays the Tentative Action Result showing just the two shipments.
  3. Click Execute to process, if the Tentative Action Result looks good. The results page will open.
    OR
    Click Cancel to stop the swap process.
