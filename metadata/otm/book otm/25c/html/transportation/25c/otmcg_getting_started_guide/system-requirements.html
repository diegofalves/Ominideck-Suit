System Requirements

Internet Browser Support

Note: The product is certified with the latest version of Chrome and Edge available at the time of release. Because of Chrome/Edge's policy of automatically updating itself whenever a new release is available, issues that occur in newer versions of the browser that weren't present in the certified version might require the use of an alternate browser until some future, scheduled release, when product can once again certify against the latest browser version.
  • Firefox ESR 128
  • Google Chrome Version 136
  • Microsoft Edge 136
  • Safari 18 (Mac OS)
Note: All supporting software is subject to patch level changes.

Mobile Requirements

  • iOS Devices: Safari
  • Android Devices: Chrome
