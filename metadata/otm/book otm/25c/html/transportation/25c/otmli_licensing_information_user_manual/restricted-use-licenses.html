Restricted Use Licenses

Oracle Transportation Management

Subproduct: Transportation Operational Planning; Fleet Management; Logistics Network Modeling

Restricted Use Licenses

  • A license to Oracle Transportation Management and the listed subprojects includes a restricted use license to Oracle Fusion Middleware HTTP Server solely to host Oracle Transportation Management and the listed subproducts.
  • A license to Oracle Transportation Management and the listed subprojects includes a restricted use license to Oracle Database solely to host the schema or objects created by the Oracle Transportation Management installer.
  • A license to Oracle Transportation Management and the listed subprojects includes a restricted use license to Oracle Analytics Server (OAS) or Oracle Analytics Cloud (OAC) solely to report on the schema and objects provided as part of Oracle Transportation Management.
