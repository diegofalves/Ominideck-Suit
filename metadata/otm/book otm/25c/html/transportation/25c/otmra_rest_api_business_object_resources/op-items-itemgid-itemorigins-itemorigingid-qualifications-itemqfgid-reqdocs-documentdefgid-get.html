Get a Standard Document

get

/items/{itemGid}/itemOrigins/{itemOriginGid}/qualifications/{itemQfGid}/reqdocs/{documentDefGid}

Request

Path Parameters
  • documentDefGid(required): string
    Resource ID
  • itemGid(required): string
    Resource ID
  • itemOriginGid(required): string
    Resource ID
  • itemQfGid(required): string
    Resource ID
Query Parameters
  • expand: string
    Comma-delimited string of child resource names that you want to expand so that the contents return with the parent in the same GET request.
  • fields: string
    Comma-delimited string of field names that you wanted returned in a GET request.

There's no request body for this operation.
