Processing Changes

This section describes the XML interface changes introduced in Oracle Transportation Management version 21A that affect the schema documents as a whole. A description of each element and attribute changes are covered in section 3 and section 4. For a complete description of the meaning and use of all interfaces refer to the integration online documentation.

Primary schema changes in 21A release came from an effort to support strict schema validation. Some key common changes include

  • Primary key element(s) (GID/Sequence) are now optional where an Integration Saved Query is supported
  • Weight and Volume are now optional in a WeightVolumeType
  • Length, Width and Height are now optional in a LengthWidthHeightType
  • GLogDate is now optional in the GLogDateTimeType
