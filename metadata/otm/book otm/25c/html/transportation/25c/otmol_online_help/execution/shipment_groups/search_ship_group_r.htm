SmartLinks

From the Shipment Group Manager results page, right-click on any ID to launch the following SmartLink.

SmartLink

Displays

Documents

Displays documents associated with the shipment group.

Full Audit

Audit Trails results page displaying all changes that were made to the shipment group. You can also perform a limited audit using the appropriate action above.

View Related Appointments

Displays appointments associated with the shipment group.

View Related Buy Shipments

Displays the buy shipments associated with the shipment group.

View Related Invoices

Displays the invoices associated with the shipment group.

View Related Order Movement

Displays the order movements associated with the shipment group.

View Related Order Releases

Displays the order releases of all the shipments associated with the shipment group.

View Related Reference Numbers

Displays the reference numbers of all the orders and all the shipments associated with the shipment group.

View Related Remarks

Displays the remarks of all the orders and all the shipments associated with the shipment group.

View Related Sell Shipments

Displays the sell shipments associated with the shipment group.

View Related Trade Transactions

Displays the trade transactions associated with the shipment group.

View Tracking Events

Displays events associated with the selected shipment group.
