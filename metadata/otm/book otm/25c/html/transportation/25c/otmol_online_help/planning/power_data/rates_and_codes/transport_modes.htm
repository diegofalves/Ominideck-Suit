Transport Modes

This page is accessed via:

  • Shipment Management > Power Data > General > Transport Modes
  • Master Data > Power Data > General > Transport Modes

You can define new modes of transportation here or use the public transportation modes. The transport mode is vital in defining a rate offering, service provider, and itinerary. It can be used to differentiate between goods from different orders that can be consolidated into one shipment for one or more legs of the trip. The transport mode is assigned to a shipment during shipment planning.

A transport mode can also be assigned to Transport Mode Profiles which are used to indicate compatibility for a group of transport modes. For example, when you create an itinerary, you can assign a mode profile that includes several transport modes thereby making the itinerary compatible for shipment planning for the modes specified in the profile.

Adding a Transport Mode

  1. Enter a Transport Mode ID.
  2. Enter a Transport Mode Name.
  3. Select a Transport Mode Qualifier to distinguish the transportation mode as either standard, hazardous, or both.
  4. Enter a Conditional Booking Profile ID to associate a conditional booking mode with the transport mode.
  5. The Color drop-down indicates what color the transport mode will be when you plot related shipments.
  6. The Transport Handling Unit Profile GID is used to indicate the transportation handling units whose use should be considered during the shipment ship unit building process which is turned on when you select the Allow Repack checkbox.
  7. The Allow Remark/Reference Number Visibility checkbox controls whether remarks/reference numbers on a primary shipment can be accessed by related shipments of this transport mode when running the View Related Remarks or View Related Reference Number SmartLinks on the related shipments.
  8. Select a mode type from the Mode Type drop-down list. Select a mode type of TL if you want to use ship unit repack for multi-stop shipments. If any other mode type is specified, ship units will not be repacked for multi-stop shipments.
  9. Select the Allow Repack checkbox Â if you want the selected mode to be considered during repack. When this is selected you can specify if cost should be considered during repack by selecting the Consider Cost During SSU Repack checkbox.
  10. The Consider Cost During SSU Repack option controls whether cost will be a consideration during the shipment ship unit building process. It is only selectable when the Allow Repack checkbox is selected.
  11. If applicable, enter an X12 Format Code.
  12. Select a Domain Name.
  13. Click Finished.
