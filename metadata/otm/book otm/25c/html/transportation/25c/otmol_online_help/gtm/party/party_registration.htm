Licenses and Registrations

Registration

This page is accessed via Licenses and Registrations > Registrations.

This page lists registration information and provides a means of recording new registration information.

New Registration

  1. Enter an ID for the registration in the RegistrationÂ ID field.
  2. Enter a jurisdiction ID in the Jurisdiction ID field.
  3. Enter a regime ID in the Regime ID field.
  4. Enter the domain from the Domain Name drop-down list.
  5. Enter a registration category ID in the Registration Category ID field.
  6. Enter a registration type ID in the Registration Type field.
  7. Enter a registration code in the Registration Code ID field.
  8. Enter a registration number in the Registration Number field.
  9. Enter a registration name in the Registration Name field.
  10. Enter an address in the Registered Address field.
  11. Enter a product classification type in the Product Classification Type ID field.
  12. Enter a classification code in the Product Classification Code field
  13. Use the remaining fields (Effective Date, Expiration Date, Requested Date, Approval Date, and Renewal Date) to record any additional information.
  14. Use the Active and Template options to indicate if the registration is active or if it is intended to be a template.

Quantity

  1. Select a quantity qualifier from the Quantity Qualifier ID drop-down list.
  2. Enter a value in the Quantity field.
  3. Select a unit of measurement from the UOM drop-down list.
  4. Click Save.

Value

  1. Select a value qualifier from the Value Qualifier ID drop-down list.
  2. Enter a value in theÂ Value field and designate a currency.
  3. Click Save.

Involved Parties

  1. Select the type of involved party from the Involved Party Qualifier ID drop-down list.
  2. Enter the contact name in the Involved Party field.
  3. Click Save.

Reference Numbers

Use the reference numbers section to add information (including license information) you wish to associate with this party.

  1. Select a party registration reference number qualifier ID from the Reference Number Qualifier ID drop-down list.
  2. Enter a reference number in the Reference Number field.
  3. Click Save.

Remarks

Use the remarks section to add text you wish to associate with this party.

  1. Select a remark qualifier ID from the Remark Qualifier ID drop-down list.
  2. Enter the remark in the Remark Text field.
  3. Click Save.

Date

  1. Select a date qualifier from the Date Qualifier ID drop-down list.
  2. Enter the registration date to the Registration Date field.
  3. Click Save.

Commodity

  1. Enter an ID in the Commodity ID field.
  2. Click Save.

User-defined Commodity

  1. Enter an ID in the User-defined Commodity ID field.
  2. Click Save.

Item

  1. Enter an ID in the Item ID field.
  2. Click Save.

Related Licenses

If there are any licenses related to this registration, use this section to associate them with the registration.
  1. Enter an ID in the License ID field.
  2. Click Save.

Related Registrations

If there are any other registrations related to this registration, use this section to associate them with the registration.
  1. Enter an ID in the Related Registration ID field.
  2. Click Save.
  3. Click Finished.
