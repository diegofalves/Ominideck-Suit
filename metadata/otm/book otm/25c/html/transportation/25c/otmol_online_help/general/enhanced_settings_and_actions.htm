Configuration and Administration

Settings and Actions

This page is accessed via the User (Settings and Actions) drop-down in the Unified Global Header.

You see the following buttons:

  • Sign Out logs you out of Oracle Transportation Management. You should always use this link to log out instead of closing the browser.
  • Save and Close saves any changes you made to user role or tracking and closes the pop-up.
  • Cancel closes the pop-up without saving your changes.
  • Expand All/Collapse All () expands or compacts the contents of the window.

The page has the following sections: User Details, System Details, Trouble Shooting, and Administration.

User Details

This section displays details about the user logged into OTM. To the right of the User Details sub-heading, you see the user name (prefixed by the domain name) of the user logged into OTM.

This section also displays details about the user and account policy information for that user which includes:

  • User Role: A user with multiple or master role assignments can change roles by selecting a role from the User Role drop-down list and clicking Save and Close. This changes the role that you are using to view OTM/GTM.
  • Nickname
  • Domain
  • Account Policy ID
  • Password Expiration Date

System Details

This section displays system configuration information:

  • software version
  • weekly patch version
  • database edition
  • analytics type: If glog.oci.isOAC is set to true, the analytics type is Oracle Analytics Cloud (OAC); otherwise, the type is Oracle Analytics Server (OAS).

You can configure explanations via the Tracking Mode and Tracking Level drop-down lists.

You can also turn on Tracking Diagnostics for a user's session. For Diagnostics andÂ graphical diagnostics to be enabled, two checks must be in place. The glog.otmdiag.log property must be on (it is on (true) by default). Also, either the DiagnosticÂ user preferences or the Tracking Diagnostic option on the Settings and Actions page must be ON. If you enable diagnostics this way, you must turn it on each time you log in since it will be automatically turned off when you log out.

To hide this section for specific users, manage user access for Settings and Actions.

Trouble Shooting

This section contains the following links:

  • System Log: Click to view the System Log search page.
  • Integration Log: Click to view the Integration Log search page.
  • Diagnostics: Click to view the View Diagnostics search page.

To hide this section for specific users, manage user access for Settings and Actions.

Note: When you click on a link, the new page replaces the main Oracle Transportation Management page.

Administration

This section contains the following links:

  • Change Password: Click to change your password. You can only change your password if you are logged into Oracle Transportation Management as a .ADMIN user.
  • View User Access: Click to see your current access to user preferences, user menus, saved queries/searches, screen sets, and field screen sets, etc.
  • User Preference: Click to open the user preference assigned to the user logged in Oracle Transportation Management.
  • Recurring Processes: Click to open the Recurring Processes Finder.

To hide this section for specific users, manage user access for Settings and Actions.

Note: When you click on a link, the new page replaces the main Oracle Transportation Management page.
