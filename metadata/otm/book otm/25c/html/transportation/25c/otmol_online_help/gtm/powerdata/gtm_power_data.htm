GTM Power Data

This page is accessed via Master Data > Power Data.

Power Data represents setup or configuration data that you should define early on in your Global Trade Management (GTM) implementation. In many cases, the definition of Power Data affects the way in which Global Trade Management functions. For example, many of the Power Data options directly impact how you work with parties, products, screening, and so on.
