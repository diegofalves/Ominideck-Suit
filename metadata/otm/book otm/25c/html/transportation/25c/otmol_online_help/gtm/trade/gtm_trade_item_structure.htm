GTM Trade Item Structure

This page is accessed via Master Data > Trade Item Structures.

Use this page to manage components of imported/exported items by listing Bill of Materials components so they can be accurately screened for compliance.

Unlike individual trade items, trade item structures include component items, for example a tool set consisting of several different types of wrenches and hand tools. The tool set itself cannot be screened for compliance; however, each individual item in the set will have a country of origin, a classification code, etc.

Additionally, this page provides the capability to define specific to generic trade item structure based on need, using the combination of product classification type and product classification code.

Identification

Use this section to record basic identification information about the trade item structure.

  1. Enter an ID for the structure in the Item Structure ID field. This is the ID that will be referenced by Global Trade Management when it refers to this structure.
  2. Enter the name of the structure in the Structure Name field.
  3. Select the domain in which this structure will be active from the Domain Name drop-down list.
  4. Enter a description of the structure in the Description field.
  5. Select the Active option to designate the trade item structure as active.
  6. Use the Effective Date and Expiration Date fields to define a specific time window for this trade item structure to be usable.
  7. Enter a country code in the Default Country of Origin field. This country code should pertain to the entire trade item structure since it is possible that some of the components may have a different country of origin.
  8. Enter a value in the Alternate Designator field.Â This links the parent and the child Bill of Material and helps to identify the alternate Bill of Material.
  9. Enter an ID in the Product Classification Type ID field.
  10. Enter a product code in the Production Classification Code field. Remember, this is the classification code for the trade structure, not one of the components.
  11. Enter the item ID in the Item ID field.
  12. Enter the Inventory Organization associated with this trade structure.
  13. Enter the party associated with the structure in the Partner field.
  14. Enter the party site associated with the structure in the Partner Site field.
  15. Enter a quantity and a unit of measure for the quantity in the Quantity and Quantity UOM fields, respectively, to ensure proper calculation of Bill of Material components.
    Note:Â You can configure the values displayed in the Quantity UOM drop-down list via the propertyÂ gtm.structure.QuantityUOM.displayUOMTypesForQuantity.

Trade Structure Components

Use this section to describe the various component items of the trade item structure. The information that you see here is read-only.

To add a new structure component, click New Structure Component. You can enter multiple components.

Involved Parties

Use this section to list the involved parties for this trade item structure.

  1. Enter an Involved Party Qualifier ID.
  2. Enter an Involved Party.
  3. Click Save. You can enter multiple involved parties.

The remainder of the information is read-only.

Values

Use this section to add information about values to the trade item structure. You can add as many value qualifiers as you need.

Adding a Value Qualifier:

  1. Select a value qualifier from the Value Qualifier ID drop-down list.
  2. Enter a value in the Value field.
  3. Select a currency from the drop-down list.
  4. Select an exchange rate from Â the Exchange Rate ID drop-down list.
  5. Enter the date for which the exchange rate is to be applied in the Exchange Rate Date field.
  6. Click Save.

User-defined Code

Use this section to add user-defined codes to the item structure.

  1. Select a user-defined code category from the Category drop-down list.
  2. Select a user-defined code type from the Type ID drop-down list.

    Note: The available selections are determined by your choice of category in the previous step.

  3. Select a user-defined code value from the Type Code drop-down list.

    Note: The available selections are determined by your choice of type in the previous step.

  4. Click Save.

Remarks

Use this section to list any remarks you wish to accompany this trade item structure.

  1. Enter a Remark Qualifier ID.
  2. Enter Remark Text.
  3. Click Save.

Reference Numbers

Use this section to add any reference numbers that may be applicable.

  1. Enter a Reference Number Qualifier ID.
  2. Enter a Reference Number.
  3. Click Save.
