Service Provider Profiles

This page is accessed via Contract and Rate Management > Power Data > General > Service Provider Profiles.

Service Provider Profiles allow you to identify and define groups of service providers that have something in common using Service Provider Profiles. Profiles can be used to include or exclude service providers from results for queries about rates, itineraries, and shipments. For example, if there is a Service Provider Profile on an itinerary, Oracle Transportation Management knows that only service providers included in the Service Provider Profile are valid for carrying the shipment.

Example: Service Provider Rule

  • Order Bundle 1 UPS
  • Order Bundle 2 UPS or FedEx
  • Order Bundle 3 FedEx

Result

  • Order Bundle 1 and 2 are compatible
  • Order Bundle 2 and 3 are compatible
  • Order Bundle 1 and 3 are not compatible

Service Provider Profile

Creating a service provider profile:

  1. Enter a Service Provider Profile ID.
  2. Enter a Service Provider Profile Name.
  3. If you select the Compatible checkbox, Oracle Transportation Management includes the service providers in the profile when searching in other areas of Oracle Transportation Management. If you clear the checkbox, Oracle Transportation Management excludes the service providers from searches in other areas of Oracle Transportation Management.

    For example, you may create a service providers called site_delivery to indicate that the service providers in this profile can deliver goods to a construction site. Because you want the service providers to be included in any searches that require delivery to a construction site, the Compatible checkbox would be marked. You can then create a second profile called no_site_delivery. This second profile may consist of service providers who do not offer delivery to a constructions site. In this instance, the Compatible checkbox would be cleared. Depending on whether the Compatible checkbox is marked, the service providers in the profile would be included or excluded from any searches in other areas of Oracle Transportation Management (such as Rate Manager or Itinerary Manager).

    Note: The Compatible checkbox is not considered by Sourcing.

  4. Enter a Service Provider ID.
  5. Click Save for each service provider IDs you selected.
  6. Click Finished.
