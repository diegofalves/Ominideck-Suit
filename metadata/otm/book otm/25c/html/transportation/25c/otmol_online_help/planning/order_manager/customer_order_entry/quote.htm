Rate Quote

This page is accessed via Order Management > Order Release > Order Release. Click the Constraints tab and then click Get Buy Rate or Get Sell Rate.

The Rate Quote page displays the results of a rate quote inquiry for the order release along with the valid service providers that are capable of handling the transportation requirements for the order release.

Note: The information returned is dependent on how the QUOTE USES RIQ ROUTE parameter is set.

The information entered in the following Order Release Manager fields is used to filter the rate lookup:
  • Service Provider
  • Transport Mode
  • Source Location
  • Destination Location
  • Flex Commodity Qualifier
  • Equipment Group ID
  • Accessorial Codes
  • Special Service Codes
  • Available dates
  • Delivery dates
  • Fixed Buy/Sell Rate Offering
  • Itinerary
  • Ship Unit or Ship Unit Lines
    • Flex Commodity Qualifier
    • Flex Commodity Code
    • Weight
    • Length
    • Width
    • Volume
    • Ship Unit Count
    • Transport Handling Unit

Note: If no itinerary or rate offering is found, a window will show you the locations used for the search and display a Refine Inquiry button which will take you back to the Constraints tab where you can make changes.

Selecting a Rate Offering

The following appears for each shipping option:

  • Service Provider ID:Â if you click the service provider ID link, the service provider window will open.
  • Service Provider Name: displays the name of the service provider.
  • Rate Offering: represents the specific contract for transportation rates from the service provider. If you click the rate, the Rate Offering window will open.
  • Mode: method used to transport the order release, for example, truckload (TL), less than truckload (LTL), and so on.
  • Delivery Time: displays the estimated delivery date/time that the order will reach its destination.
  • Transit Time: displays the total duration (according to the User Preferences setting for duration) that it takes to travel from the source to the destination.
  • Distance: displays the calculated distance between the source and destination location.
  • Cost: calculated based on the order release characteristics and the Rate Offering information that has been defined for the service provider.
  • CO2 Emissions:Â display the CO2 emissions calculated during shipment planning.Â 
  • CO2e Emissions:Â display the CO2e emissions calculated during shipment planning.

The legend icons show the optimal service provider (fastest transit time and cheapest cost), service providers that can meet the delivery/service time requirements, and service providers that cannot meet the delivery/service time requirements,

  1. Select the rate you wish to use by clicking the option to the left of the rate.

    Note: If you have set the QUOTE USES RIQ ROUTE parameter to True, you will see itineraries on this page instead of quotes.

    Note: If a rate has been designated a preferred rate, a lightning bolt icon will be displayed to the left of the rate information.

  2. Click Save. The rate information for the selected rate will be populated onto the Constraints tab.

Refining an Inquiry:

Click Refine Inquiry to return to the Order Release Constraints page and modify the order information. Then request another quote. Also, use this option if you select a service provider by mistake and want to change your selection.
