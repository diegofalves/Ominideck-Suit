About Enhanced Workbench Layouts

Note: All workbenches available from the PUBLIC menus will automatically open as enhanced workbenches.

A workbench layout is a multi-pane layout that can contain one or more of the following:

  • tables
  • maps
  • a Gantt chart
  • manager layouts
  • 3D load configuration viewers
  • analytics projects and reports
  • shipment timelines
  • diagrams (for routing networks only)
  • diagnostics

Each layout can have multiple regions, with each region containing panes that display different, but related, information. For example, you might have a workbench containing a layout that has three panes: a list of buy shipments, a map, and a driver or work assignment Gantt chart. The map would display the routes of the buy shipments, order movements, order releases, and drivers or work assignments, while the Gantt chart would show the buy shipments and drivers or work assignments.

The workbench designer allows you to create and edit workbench layouts. See Enhanced WorkbenchÂ for more information.

You can open both legacy Workbench or Enhanced Workbench layouts. However, the legacy Workbenches are read-only. To convert a legacy Workbench layout to an Enhanced Workbench layout, use the Workbench Manager.

You can setÂ User Preferences, calledÂ Home Workbench Layout ID, to use a certain enhanced workbench layout as your landing page when you sign in.

OTM and GTM include a variety of PUBLIC layouts:

  • Sample workbench layouts which are examples of how you can use workbench layouts

Using a Workbench Layout

Select a layout from the Workbench Layout drop-down list. This list will show both enhanced and legacy layout IDs. The legacy layout IDs are marked with an asterisk. Any legacy layouts that you view in the Enhanced WorkbenchÂ uses all of the enhanced layout icons. While these layouts can be viewed in the Enhanced Workbench, they cannot be edited. In order to edit a legacy layout, you can convert legacy layouts to enhanced layouts using the Workbench Manager actions.

Once the layout loads, depending on how the layout is defined, different panes may be available. Options are available for multiple component types. If you select an object in one pane of the layout, it will highlight in the other panes in the layout. For example, if you select a shipment in a table, then it will appear selected in the map pane and/or in the Gantt pane.

(Last/Next Auto Refresh Time): This icon only appears if Auto Refresh is turned on for the selected workbench layout. When you hover the mouse pointer over this image, you see the Last Refresh Time (if there was one) and Next Refresh Time. This icon cannot be clicked only hovered over. The data in this hover updates when Auto Refresh runs or when you click Refresh All Data on an auto-refresh layout. This icon appears in one place which is the view only version of the layout.
