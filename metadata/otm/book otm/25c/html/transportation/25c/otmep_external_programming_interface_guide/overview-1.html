Overview

The intent of this document is to provide documentation for the Web Service Interfaces that are provided for customers or third parties to extend existing Oracle Transportation and Global Trade Management Cloud functionality. Oracle Transportation and Global Trade Management Cloud connects as a client to these web services implemented by the customers or third parties conforming to the WSDL provided.

The Oracle Transportation and Global Trade Management Cloud products expose interfaces to support integration with external application in the areas of distance, service time and cost calculation. This document will describe the following:

  • External Distance Engine (EDE) Interface
  • External Service Engine (ESE) Interface
  • External Rating Engine (ERE) Interface (using SOAP and REST web service)
